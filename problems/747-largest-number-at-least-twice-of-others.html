Largest Number At Least Twice of Others

O(n) O(1) 52.5% AC
Mô tả
Mảng nums. Kiểm tra có phần tử nào lớn gấp ít nhất 2 lần mọi phần tử khác không. Có thì trả index lớn nhất, không thì -1.
Ví dụ
Ví dụ 1
Innums = [3,6,1,0]
Out1
Ví dụ 2
Innums = [1,2,3,4]
Out-1
Ví dụ 3
Innums = [1]
Out0
Ý tưởng
Tìm max1, max2 trong một pass. Nếu max1>=2*max2 return index max1 cuối cùng.

int dominantIndex(int* nums, int n) {
    int mx = nums[0], mx2 = 0, idx = 0;
    for (int i = 0; i < n; i++) {
        if (nums[i] > mx) { mx2 = mx; mx = nums[i]; idx = i; }
        else if (nums[i] > mx2) mx2 = nums[i];
    }
    return mx >= 2 * mx2 ? idx : -1;
}
class Solution:
    def dominantIndex(self, nums: List[int]) -> int:
        mx = max(nums)
        if mx < 2 * max(x for x in nums if x != mx): return -1
        return nums.index(mx)
class Solution {
public:
    int dominantIndex(vector<int>& nums) {
        int mx=*max_element(nums.begin(),nums.end()), idx=-1, mx2=0;
        for(int i=0;i<(int)nums.size();i++){
            if(nums[i]==mx) idx=i;
            else mx2=max(mx2,nums[i]);
        }
        return mx>=2*mx2?idx:-1;
    }
};
