Reach a Number

O(1) O(1) 45.1% AC
Mô tả
Đứng tại 0, mỗi bước +1 hoặc -1. Sau |target| bước đến target. Trả số bước tối thiểu (luôn đi đủ |target| bước).
Ví dụ
Ví dụ 1
Intarget = 2
Out3
Ví dụ 2
Intarget = 3
Out2
Ví dụ 3
Intarget = 1000
Out32
Ý tưởng
Tăng k: tổng 1+2+..+k >= |target| và (sum-target) chẵn để flip một nửa bước thành -.
Mã nguồn
solution.c C99 
int reachNumber(int target) {
    target = target < 0 ? -target : target;
    for (long long k = 1;; k++) {
        long long s = k * (k + 1) / 2;
        if (s >= target && (s - target) % 2 == 0) return (int)k;
    }
}
class Solution:
    def reachNumber(self, target: int) -> int:
        target = abs(target)
        k = 0
        while True:
            s = k * (k + 1) // 2
            if s >= target and (s - target) % 2 == 0:
                return k
            k += 1
