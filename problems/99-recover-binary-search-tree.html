Recover Binary Search Tree

O(n) O(h) 59.7% AC
Mô tả
BST với đúng hai nút bị hoán đổi nhầm giá trị. Sửa in-place (chỉ đổi giá trị hai nút), không thay cấu trúc.
Ví dụ
Ví dụ 1
Inroot = [1,3,null,null,2]
Out[3,1,null,null,2]
Ví dụ 2
Inroot = [3,1,4,null,null,2]
Out[2,1,4,null,null,3]
Ý tưởng
Inorder: tìm hai cặp prev>cur (first lần đầu, second cập nhật). Hoán đổi first và second.val. Morris hoặc stack O(1) space.
Mã nguồn
solution.c C99 
/* recoverTree — inorder tìm 2 nút; xem C++ */
class Solution:
    def recoverTree(self, root: Optional[TreeNode]) -> None:
        first = second = prev = None
        st, cur = [], root
        while cur or st:
            while cur:
                st.append(cur); cur = cur.left
            cur = st.pop()
            if prev and prev.val > cur.val:
                if not first: first = prev
                second = cur
            prev = cur; cur = cur.right
        first.val, second.val = second.val, first.val
class Solution {
public:
    void recoverTree(TreeNode* root) {
        TreeNode *first=nullptr, *second=nullptr, *prev=nullptr;
        stack<TreeNode*> st; TreeNode* cur=root;
        while (cur||!st.empty()) {
            while (cur) { st.push(cur); cur=cur->left; }
            cur=st.top(); st.pop();
            if (prev && prev->val > cur->val) {
                if (!first) first=prev;
                second=cur;
            }
            prev=cur; cur=cur->right;
        }
        swap(first->val, second->val);
    }
};
