Binary Tree Level Order Traversal

O(N) O(N) (Kích thước Hàng đợi chứa tầng lá) 72.7% AC
Mô tả
Duyệt cây nhị phân theo từng tầng từ trên xuống dưới, từ trái qua phải (BFS Tree Traversal) và trả về mảng 2D.
Ví dụ
Ví dụ 1
Inroot = [3,9,20,null,null,15,7]
Out[[3],[9,20],[15,7]]
Ý tưởng
Sử dụng cấu trúc dữ liệu Hàng đợi (Queue) để thực hiện duyệt theo chiều rộng (BFS). Ta đẩy nút gốc vào Queue. Trong vòng lặp ngoài, mỗi lượt ta đếm kích thước hiện tại của Queue `size` (tương ứng số lượng nút trong tầng). Vòng lặp trong chạy đúng `size` lần để rút nút ra, lưu giá trị, và đẩy các con trái/phải tương ứng vào Queue.

// Requires standard dynamic queue allocation for levels traversal representation.
# Solution in Python (Binary Tree Level Order Traversal)
from collections import deque
class Solution:
    def levelOrder(self, root: Optional[TreeNode]) -> List[List[int]]:
        if not root:
            return []
        res, queue = [], deque([root])
        while queue:
            level = []
            for _ in range(len(queue)):
                node = queue.popleft()
                level.append(node.val)
                if node.left: queue.append(node.left)
                if node.right: queue.append(node.right)
            res.append(level)
        return res
class Solution {
public:
    vector<vector<int>> levelOrder(TreeNode* root) {
        if (!root) return {};
        vector<vector<int>> res;
        queue<TreeNode*> q;
        q.push(root);
        while (!q.empty()) {
            int size = q.size();
            vector<int> level;
            for (int i = 0; i < size; i++) {
                TreeNode* curr = q.front(); q.pop();
                level.push_back(curr->val);
                if (curr->left) q.push(curr->left);
                if (curr->right) q.push(curr->right);
            }
            res.push_back(level);
        }
        return res;
    }
};
