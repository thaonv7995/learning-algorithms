Reverse Bits

O(1) O(1) 68.4% AC
Mô tả
Đảo ngược 32 bit của số nguyên không dấu n. Trả kết quả dạng unsigned.
Ví dụ
Ví dụ 1
Inn = 43261596
Out964176192
Ví dụ 2
Inn = 2147483644
Out1073741822
Ý tưởng
Lặp 32 lần: res = (res<<1) | (n&1); n >>= 1. Hoặc swap bit pairs/groups (divide conquer).
Mã nguồn
solution.c C99 
uint32_t reverseBits(uint32_t n) {
    uint32_t res = 0;
    for (int i = 0; i < 32; i++) {
        res = (res << 1) | (n & 1u);
        n >>= 1;
    }
    return res;
}
class Solution:
    def reverseBits(self, n: int) -> int:
        res = 0

            res = (res << 1) | (n & 1)
            n >>= 1
