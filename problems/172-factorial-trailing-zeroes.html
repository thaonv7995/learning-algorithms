Factorial Trailing Zeroes

O(log n) O(1) 46.7% AC
Mô tả
Cho n, tính số chữ số 0 ở cuối của n! (trailing zeroes). Không tính trực tiếp factorial vì n lớn.
Ví dụ
Ví dụ 1
Inn = 3
Out0
Ví dụ 2
Inn = 5
Out1
Ví dụ 3
Inn = 0
Out0
Ý tưởng
Mỗi trailing zero = một cặp nhân tử 2×5; trong n! luôn thừa số 2 hơn 5. Đếm số lần 5 chia n: n/5 + n/25 + n/125 + …
Mã nguồn
solution.c C99 
int trailingZeroes(int n) {
    int cnt = 0;
    for (long long p = 5; p <= n; p *= 5) cnt += n / p;
    return cnt;
}
class Solution:
    def trailingZeroes(self, n: int) -> int:
        cnt = 0
        while n:
            n //= 5
            cnt += n
        return cnt
class Solution {
public:
    int trailingZeroes(int n) {

        return cnt;

Math O(log n) O(1)

  2. 2Duyệt chính — luôn thừa số 2 hơn 5.
  3. 3Cập nhật — Đếm số lần 5 chia n: n/5 + n/25 + n/125 + …
