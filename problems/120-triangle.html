O(n²) O(n) 59.9% AC
Mô tả
Cho tam giác số (hàng i có i+1 phần tử). Tìm tổng nhỏ nhất trên đường đi từ đỉnh xuống đáy; mỗi bước chỉ xuống ô liền kề bên dưới trái hoặc phải.
Ví dụ
Ví dụ 1
Intriangle = [[2],[3,4],[6,5,7],[4,1,8,3]]
Out11
Ví dụ 2
Intriangle = [[-10]]
Out-10
Ý tưởng
DP bottom-up: dp = hàng cuối. Duyệt từ hàng n-2 lên 0: dp[j] = triangle[i][j] + min(dp[j], dp[j+1]). Trả dp[0].
Mã nguồn
solution.c C99 
/* DP bottom-up 1D min — xem C++ */
class Solution:
    def minimumTotal(self, triangle: List[List[int]]) -> int:
        dp = triangle[-1][:]
        for i in range(len(triangle) - 2, -1, -1):
            for j in range(len(triangle[i])):
                dp[j] = triangle[i][j] + min(dp[j], dp[j + 1])
        return dp[0]
class Solution {
public:
    int minimumTotal(vector<vector<int>>& tri) {
        vector<int> dp = tri.back();
        for (int i = (int)tri.size() - 2; i >= 0; i--)
            for (int j = 0; j <= i; j++)
                dp[j] = tri[i][j] + min(dp[j], dp[j + 1]);
        return dp[0];
    }
};
