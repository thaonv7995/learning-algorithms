Remove Nth Node From End of List

O(n) O(1) 51.6% AC
Mô tả
Cho head của linked list đơn và s integer n ≥ 1. Xóa nút thứ n tính từ cuối list (1-indexed từ cuối) và trả head mới.
Ví dụ
Ví dụ 1
Inhead = [1,2,3,4,5], n = 2
Out[1,2,3,5]
Ví dụ 2
Inhead = [1], n = 1
Out[]
Ý tưởng
Dummy head + two pointers: đưa fast trước slow n+1 bước, rồi cùng tiến đến khi fast=null. slow.next là nút cần xóa.
Mã nguồn
solution.c C99 
/* Dummy + two pointers — xem C++ */
class Solution:
    def removeNthFromEnd(self, head: Optional[ListNode], n: int) -> Optional[ListNode]:
        dummy = ListNode(0, head)
        fast = slow = dummy
        for _ in range(n + 1):
            fast = fast.next
        while fast:
            fast = fast.next
            slow = slow.next
        slow.next = slow.next.next
        return dummy.next
class Solution {
public:
    ListNode* removeNthFromEnd(ListNode* head, int n) {
        ListNode dummy(0, head);
        ListNode *fast = &dummy, *slow = &dummy;
        for (int i = 0; i <= n; ++i) fast = fast->next;
        while (fast) { fast = fast->next; slow = slow->next; }
        slow->next = slow->next->next;
        return dummy.next;
    }
};
