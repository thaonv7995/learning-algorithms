LC #19

Remove Nth Node From End of List

Medium Linked List
Đã đồng bộ Sandbox
Mô Tả Bài Toán
Xóa nút thứ n tính từ cuối danh sách liên kết đơn lên và trả về nút gốc của danh sách mới.
Ví Dụ Minh Họa
Ví dụ 1:
Đầu vào: head = 1->2->3->4->5, n = 2
Kết quả: 1->2->3->5
Ý Tưởng & Tư Duy Giải Quyết
Áp dụng kỹ thuật Hai con trỏ cách khoảng (Two Pointers - Fast and Slow). Ta sử dụng nút giả `dummy` trỏ tới `head`. Thiết lập hai con trỏ `first` và `second` ở nút `dummy`. Dịch chuyển `first` đi trước n + 1 bước. Sau đó, dịch chuyển đồng thời cả hai con trỏ từng bước một cho đến khi `first` gặp NULL. Khi đó, `second` sẽ dừng ngay trước nút cần xóa. Ta thực hiện `second->next = second->next->next` để ngắt kết nối nút cần xóa.
solution.c C99 
// Solution in C (Remove Nth Node From End of List)
struct ListNode* removeNthFromEnd(struct ListNode* head, int n) {
    struct ListNode dummy;
    dummy.val = 0;
    dummy.next = head;
    struct ListNode* first = &dummy;
    struct ListNode* second = &dummy;
    for (int i = 0; i <= n; i++) {
        first = first->next;
    }
    while (first != NULL) {
        first = first->next;
        second = second->next;
    }
    second->next = second->next->next;
    return dummy.next;
}
# Solution in Python (Remove Nth Node From End of List)
class Solution:
    def removeNthFromEnd(self, head: Optional[ListNode], n: int) -> Optional[ListNode]:
        dummy = ListNode(0, head)
        first = second = dummy
        for _ in range(n + 1):
            first = first.next
        while first:
            first = first.next
            second = second.next
        second.next = second.next.next
        return dummy.next
class Solution {
public:
    ListNode* removeNthFromEnd(ListNode* head, int n) {
        ListNode dummy(0);
        dummy.next = head;
        ListNode* first = &dummy;
        ListNode* second = &dummy;
        for (int i = 0; i <= n; i++) first = first->next; // Di truoc n+1 buoc
        while (first) {
            first = first->next;
            second = second->next;
        }
        ListNode* toDelete = second->next;
        second->next = second->next->next; // Ngat ket noi nut
        delete toDelete; // Giai phong Heap RAM trong C++
        return dummy.next;
    }
};
Độ Phức Tạp Thuật Toán
Time Complexity O(N) Thời gian xử lý
Space Complexity O(1) Bộ nhớ phụ trợ
Phân Tích Bộ Nhớ & Pointer RAM
Kỹ thuật một lượt duyệt (One-pass) giúp giải thuật đạt tốc độ lý tưởng. Luôn ghi nhớ giải phóng (delete/free) nút bị xóa để tránh rò rỉ bộ nhớ Heap.
