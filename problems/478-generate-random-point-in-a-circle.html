Generate Random Point in a Circle

O(1) avg O(1) 42.9% AC
Mô tả
Generate uniform random point trong hình tròn radius radius tâm (x_center, y_center).
Ví dụ
Ví dụ 1
Inradius=1, x_center=0, y_center=0
Outpoints in unit disk uniform
Ví dụ 2
Inradius=0
Outalways center
Ý tưởng
Rejection: sample square [cx-r,cx+r]×[cy-r,cy+r] until inside circle; hoặc polar sqrt(U)*r.

class Solution:
    def __init__(self, radius: float, x_center: float, y_center: float):
        self.r, self.xc, self.yc = radius, x_center, y_center
    def randPoint(self) -> List[float]:
        while True:
            x = random.uniform(self.xc-self.r, self.xc+self.r)
            y = random.uniform(self.yc-self.r, self.yc+self.r)
            if (x-self.xc)**2 + (y-self.yc)**2 <= self.r**2:
                return [x, y]
