Remove Linked List Elements

O(n) O(1) 54.5% AC
Mô tả
Cho linked list và số nguyên val. Xóa mọi node có giá trị val, trả về head mới của list (head có thể thay đổi sau khi xóa).
Ví dụ
Ví dụ 1
Inhead = [1,2,6,3,4,5,6], val = 6
Out[1,2,3,4,5]
Ví dụ 2
Inhead = [], val = 1
Out[]
Ví dụ 3
Inhead = [7,7,7,7], val = 7
Out[]
Ý tưởng
Dummy node trước head; duyệt prev: nếu prev.next.val==val thì prev.next=prev.next.next, else prev=prev.next.
Mã nguồn
solution.c C99 
struct ListNode* removeElements(struct ListNode* head, int val) {
    struct ListNode dummy = {0, head};
    struct ListNode* prev = &dummy;
    while (prev->next) {
        if (prev->next->val == val)
            prev->next = prev->next->next;
        else
            prev = prev->next;
    }
    return dummy.next;
}
class Solution:
    def removeElements(self, head: Optional[ListNode], val: int) -> Optional[ListNode]:
