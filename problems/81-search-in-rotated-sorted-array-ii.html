Search in Rotated Sorted Array II

O(log n) avg, O(n) worst O(1) 40.1% AC
Mô tả
Mảng nums length n đã sort rồi rotate tại pivot không xác định; có thể trùng giá trị. Kiểm tra target có trong nums không.
Ví dụ
Ví dụ 1
Innums = [2,5,6,0,0,1,2], target = 0
Outtrue
Ví dụ 2
Innums = [2,5,6,0,0,1,2], target = 3
Outfalse
Ý tưởng
Binary search: nếu nums[lo]==nums[mid]==nums[hi] thì lo++, hi--; else xác định nửa sort (so nums[lo] vs nums[mid]) và thu hẹp như rotated I.
Mã nguồn
solution.c C99 
/* search rotated II — BS + shrink dup; xem C++ */
class Solution:
    def search(self, nums: List[int], target: int) -> bool:
        lo, hi = 0, len(nums) - 1
        while lo <= hi:
            mid = (lo + hi) // 2
            if nums[mid] == target: return True
            if nums[lo] == nums[mid] == nums[hi]:
                lo += 1; hi -= 1; continue
            if nums[lo] <= nums[mid]:
                if nums[lo] <= target < nums[mid]: hi = mid - 1
                else: lo = mid + 1
            else:
                if nums[mid] < target <= nums[hi]: lo = mid + 1
                else: hi = mid - 1
        return False
class Solution {
public:
    bool search(vector<int>& a, int t) {
        int lo=0, hi=(int)a.size()-1;
        while (lo<=hi) {
            int mid=(lo+hi)/2;
            if (a[mid]==t) return true;
            if (a[lo]==a[mid]&&a[mid]==a[hi]) { lo++; hi--; continue; }
            if (a[lo]<=a[mid]) {
                if (a[lo]<=t&&t<a[mid]) hi=mid-1; else lo=mid+1;
            } else {
                if (a[mid]<t&&t<=a[hi]) lo=mid+1; else hi=mid-1;
            }
        }
        return false;
    }
};
