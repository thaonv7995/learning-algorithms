Binary Tree Tilt

O(n) O(h) 65.9% AC
Mô tả
Tilt của node = |tổng cây con trái - tổng cây con phải|. Trả tổng tilt mọi node.
Ví dụ
Ví dụ 1
Inroot = [1,2,3]
Out1
Ví dụ 2
Inroot = [4,-9,20,null,null,15,7]
Out44
Ý tưởng
Postorder: trả tổng subtree; cộng |sumL-sumR| vào global answer.
Mã nguồn
solution.c C99 
/* Postorder subtree sum — xem C++ */
class Solution:
    def findTilt(self, root: Optional[TreeNode]) -> int:
        self.ans = 0
        def dfs(n):
            if not n: return 0
            l, r = dfs(n.left), dfs(n.right)
            self.ans += abs(l - r)
            return n.val + l + r
        dfs(root)
        return self.ans
class Solution {
    int ans = 0;
    int dfs(TreeNode* n) {
        if (!n) return 0;
        int l = dfs(n->left), r = dfs(n->right);
        ans += abs(l - r);
        return n->val + l + r;
    }
public:
    int findTilt(TreeNode* root) {
        dfs(root);
        return ans;
    }
};
