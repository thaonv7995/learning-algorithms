Perfect Number

O(sqrt n) O(1) 48.9% AC
Mô tả
Số hoàn hảo là số bằng tổng các ước dương đúng (không tính chính nó). Kiểm tra num có phải perfect number không.
Ví dụ
Ví dụ 1
Innum = 28
Outtrue
Ví dụ 2
Innum = 7
Outfalse
Ví dụ 3
Innum = 1
Outfalse
Ý tưởng
Cộng ước từ 1 đến sqrt(num); nếu i chia hết thì cộng i và num/i (trừ trùng sqrt).
Mã nguồn

/* Sum divisors sqrt — xem C++ */
class Solution:
    def checkPerfectNumber(self, num: int) -> bool:
        if num <= 1:
            return False
        s = 1
        i = 2
        while i * i <= num:
            if num % i == 0:
                s += i
                if i * i != num:
                    s += num // i
            i += 1
        return s == num
class Solution {
public:
    bool checkPerfectNumber(int num) {
        if (num <= 1) return false;
        int sum = 1;
        for (int i = 2; i * i <= num; i++) {
            if (num % i == 0) {
                sum += i;
                if (i * i != num) sum += num / i;
            }
        }
        return sum == num;
    }
};
