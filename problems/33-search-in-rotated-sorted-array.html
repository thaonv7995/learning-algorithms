Search in Rotated Sorted Array

O(log N) O(1) 44.9% AC
Mô tả
Mảng đã sort được xoay tại pivot (mọi phần tử khác nhau). Tìm target, trả index hoặc -1.
Ví dụ
Ví dụ 1
Innums = [4,5,6,7,0,1,2], target = 0
Out4
Ví dụ 2
Innums = [4,5,6,7,0,1,2], target = 3
Out-1
Ví dụ 3
Innums = [1], target = 0
Out-1
Ý tưởng
Binary search: xác định nửa [lo,mid] hay [mid,hi] còn sorted. Kiểm tra target nằm trong nửa sorted → thu hẹp, không thì nửa kia.
Mã nguồn
solution.c C99 
int search(int* nums, int n, int target) {
    int lo = 0, hi = n - 1;
    while (lo <= hi) {
        int mid = lo + (hi - lo) / 2;
        if (nums[mid] == target) return mid;
        if (nums[lo] <= nums[mid]) {
            if (target >= nums[lo] && target < nums[mid]) hi = mid - 1;
            else lo = mid + 1;
        } else {
            if (target > nums[mid] && target <= nums[hi]) lo = mid + 1;
            else hi = mid - 1;
        }
    }
    return -1;
}
class Solution:
    def search(self, nums, target):
        lo, hi = 0, len(nums) - 1
        while lo <= hi:
            mid = (lo + hi) // 2
            if nums[mid] == target: return mid
            if nums[lo] <= nums[mid]:
                if nums[lo] <= target < nums[mid]: hi = mid - 1
                else: lo = mid + 1
            else:
                if nums[mid] < target <= nums[hi]: lo = mid + 1
                else: hi = mid - 1
        return -1
