Rotated Digits

Mô tả
Đếm số x trong [1,N] mà khi xoay 180° (2↔5, 6↔9, 0/1/8 giữ) vẫn là số hợp lệ khác và khác x.
Ví dụ
Ví dụ 1

Out4
Ví dụ 2
InN = 1
Out0
Ví dụ 3
InN = 20
Out6
Ý tưởng
DFS build valid rotated numbers <= N digit by digit với map pairs; hoặc DP digit với constraint.

class Solution:
    def rotatedDigits(self, n: int) -> int:
        pairs = {0:0,1:1,8:8,2:5,5:2,6:9,9:6}
        def ok(x):
            s=str(x); r="".join(str(pairs[int(c)]) for c in s)
            return r!=s and int(r)<=n
        return sum(ok(i) for i in range(1,n+1))
class Solution {
public:
    int rotatedDigits(int n) {
        int mp[10]={0,1,-1,-1,-1,2,-1,-1,8,-1};
        auto good=[&](int x){
            string s=to_string(x), r;
            for(char c:s){ int d=c-'0'; if(d==2) r+='5'; else if(d==5) r+='2'; else if(d==6) r+='9'; else if(d==9) r+='6'; else if(d==0||d==1||d==8) r+=c; else return false; }
            return r!=s;
        };
        int ans=0; for(int i=1;i<=n;i++) if(good(i)) ans++; return ans;
    }
};
