Majority Element

O(n) O(1) 66.3% AC
Mô tả
Cho mảng nums kích thước n. Phần tử majority xuất hiện hơn ⌊n/2⌋ lần. Đề bảo đảm majority luôn tồn tại. Trả phần tử đó.
Ví dụ
Ví dụ 1
Innums = [3,2,3]
Out3
Ví dụ 2
Innums = [2,2,1,1,1,2,2]
Out2
Ý tưởng
Boyer-Moore voting: candidate + count. Gặp x==candidate thì count++; khác thì count--, count=0 thì đổi candidate. Cuối cùng candidate là majority.
Mã nguồn
solution.c C99 
int majorityElement(int* nums, int n) {
    int cand = 0, cnt = 0;
    for (int i = 0; i < n; i++) {
        if (cnt == 0) { cand = nums[i]; cnt = 1; }
        else if (nums[i] == cand) cnt++;
        else cnt--;
    }
    return cand;
}
class Solution:
    def majorityElement(self, nums: List[int]) -> int:
        cand, cnt = None, 0
        for x in nums:
            if cnt == 0:
                cand, cnt = x, 1
            elif x == cand:
                cnt += 1
            else:
                cnt -= 1
        return cand
