Max Chunks To Make Sorted

O(n) O(1) 64.2% AC
Mô tả
Permutation nums của 0..n-1 (unique). Chia max số chunk, sort từng chunk, ghép thành sorted. Trả số chunk tối đa.
Ví dụ
Ví dụ 1
Innums = [4,3,2,1,0]
Out1
Ví dụ 2
Innums = [1,0,2,3,4]
Out4
Ý tưởng
Duyệt i: end = max(end, nums[i]). Nếu end==i thì hoàn thành một chunk (+1).
Mã nguồn
solution.c C99 
int maxChunksToSorted(int* nums, int n) {
    int ans=0, end=0;
    for(int i=0;i<n;i++){ if(nums[i]>end) end=nums[i]; if(end==i) ans++; }
    return ans;
}
class Solution:
    def maxChunksToSorted(self, nums: List[int]) -> int:
        ans = end = 0
        for i, x in enumerate(nums):
            end = max(end, x)
            if end == i: ans += 1
        return ans
class Solution {
public:
    int maxChunksToSorted(vector<int>& nums) {
        int ans=0, end=0;
        for(int i=0;i<(int)nums.size();i++){
            end=max(end,nums[i]);
            if(end==i) ans++;
        }
        return ans;
    }
};
