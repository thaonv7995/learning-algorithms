Water and Jug Problem

O(log max(x,y)) O(1) 45.7% AC
Mô tả
Hai bình x và y gallon, có vòi nước vô hạn. Có thể fill, empty, pour giữa hai bình. Có đo được z gallon ở một bình không?
Ví dụ
Ví dụ 1
Inx = 3, y = 5, z = 4
Outtrue
Ví dụ 2
Inx = 2, y = 6, z = 5
Outfalse
Ý tưởng
Khả thi iff z=0 hoặc z≤max(x,y) và z chia hết gcd(x,y) (Bezout — tổ hợp tuyến tính ax+by).
Mã nguồn
solution.c C99 
static int gcdInt(int a, int b) { while (b) { int t=a%b; a=b; b=t; } return a; }
bool canMeasureWater(int x, int y, int target) {
    if (target == 0) return true;
    int g = gcdInt(x, y);
    return target <= (x > y ? x : y) && target % g == 0;
}
class Solution:
    def canMeasureWater(self, x: int, y: int, target: int) -> bool:
        if target == 0: return True
        g = gcd(x, y)
        return target <= max(x, y) and target % g == 0
class Solution {
    int gcd(int a, int b) { return b ? gcd(b, a % b) : a; }
public:
    bool canMeasureWater(int x, int y, int target) {
        if (target == 0) return true;
        int g = gcd(x, y);
        return target <= max(x, y) && target % g == 0;
    }
};
