Friend Requests II: Who Has the Most Friends

O(n log n) O(n) 62.7% AC
Mô tả
Bảng RequestAccepted(requester_id, accepter_id) — mỗi hàng là một tình bạn đã chấp nhận. Trả id và số bạn bè của người có nhiều bạn nhất (một hàng nếu hòa theo LeetCode).
Ví dụ
Ví dụ 1
InA chấp nhận B, A chấp nhận C
Outid=A, num=2
Ví dụ 2
InChỉ một cặp A↔B
OutCả A và B đều có 1 bạn
Ví dụ 3
InNhiều người cùng max
OutMột id (LIMIT 1)
Ý tưởng
UNION ALL requester_id và accepter_id thành id; GROUP BY id COUNT(*); ORDER BY num DESC LIMIT 1.
Mã nguồn
solution.c C99 
const char query[] = "SELECT id, COUNT(*) AS num FROM ( SELECT requester_id AS id FROM RequestAccepted UNION ALL SELECT accepter_id AS id FROM RequestAccepted ) t GROUP BY id ORDER BY num DESC LIMIT 1;";
# SQL
SELECT id, COUNT(*) AS num
FROM (
  SELECT requester_id AS id FROM RequestAccepted
  UNION ALL
  SELECT accepter_id AS id FROM RequestAccepted
) t
GROUP BY id
ORDER BY num DESC
LIMIT 1;
// SQL
