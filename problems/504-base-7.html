Base 7

O(log n) O(1) 54.6% AC
Mô tả
Cho số nguyên num. Trả về biểu diễn base-7 dạng chuỗi (kể cả số âm, thêm dấu trừ). num=0 trả "0".
Ví dụ
Ví dụ 1
Innum = 100
Out"134"
Ví dụ 2
Innum = -7
Out"-10"
Ví dụ 3
Innum = 0
Out"0"
Ý tưởng
Lặp lấy num % 7 và num // 7 (Python) hoặc xử lý dấu riêng với C++ % âm.
Mã nguồn
solution.c C99 
/* Chia dư base 7 — xem C++ */
class Solution:
    def convertToBase7(self, num: int) -> str:
        if num == 0:
            return "0"
        neg = num < 0
        num = abs(num)
        s = []
        while num:
            s.append(str(num % 7))
            num //= 7
        res = "".join(reversed(s))
        return "-" + res if neg else res
class Solution {
public:
    string convertToBase7(int num) {
        if (num == 0) return "0";
        bool neg = num < 0;
        long long n = abs((long long)num);
        string s;
        while (n) { s.push_back('0' + n % 7); n /= 7; }
        reverse(s.begin(), s.end());
        return neg ? "-" + s : s;
    }
};
