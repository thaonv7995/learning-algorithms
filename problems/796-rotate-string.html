Rotate String

O(n) O(n) 66.7% AC
Mô tả
Kiểm tra s có thể nhận được bằng cách xoay vòng goal (ghép goal+goal chứa s) không.
Ví dụ
Ví dụ 1
Ins = "abcde", goal = "cdeab"
Outtrue
Ví dụ 2
Ins = "abcde", goal = "abced"
Outfalse
Ví dụ 3
Ins = "aa", goal = "a"
Outfalse
Ý tưởng
Equal length và s in goal+goal (or goal in s+s).
Mã nguồn
solution.c C99 
bool rotateString(char* s, char* goal) {
    int n=strlen(s); if(n!=(int)strlen(goal)) return false;
    char* d=malloc(2*n+1); strcpy(d,s); strcat(d,s);
    int ok=strstr(d,goal)!=NULL; free(d); return ok;
}
class Solution:
    def rotateString(self, s: str, goal: str) -> bool:
        return len(s) == len(goal) and s in goal + goal
class Solution {
public:
    bool rotateString(string s, string goal) {
        return s.size()==goal.size() && (goal+goal).find(s)!=string::npos;
    }
};
