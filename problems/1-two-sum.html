Two Sum

O(N) O(N)
Mô tả
Cho một mảng số nguyên nums và một số nguyên target, hãy tìm chỉ số của hai số sao cho tổng của chúng bằng target. Giả định rằng mỗi đầu vào có duy nhất một giải pháp và bạn không được phép sử dụng cùng một phần tử hai lần.
Ví dụ
Ví dụ 1
Innums = [2, 7, 11, 15], target = 9
Out[0, 1] (Vì nums[0] + nums[1] == 9)
Ý tưởng
Sử dụng kỹ thuật Hash Map (Bảng băm) một lượt quét. Khi đi qua từng số, ta tính toán giá trị cần tìm (complement = target - nums[i]). Nếu complement đã tồn tại trong Hash Map, ta lập tức trả về chỉ số của complement và chỉ số hiện tại. Nếu chưa, ta ghi nhận giá trị hiện tại cùng chỉ số của nó vào Hash Map để các phần tử phía sau tra cứu.
Mã nguồn
solution.c C99 
// Solution in C (Two Sum)
int* twoSum(int* nums, int numsSize, int target, int* returnSize) {
    *returnSize = 2;
    int* res = (int*)malloc(2 * sizeof(int));
    for (int i = 0; i < numsSize; i++) {
        for (int j = i + 1; j < numsSize; j++) {
            if (nums[i] + nums[j] == target) {
                res[0] = i;
                res[1] = j;
                return res;
            }
        }
    }
    *returnSize = 0;
    return NULL;
}
# Solution in Python (Two Sum)
class Solution:
    def twoSum(self, nums: List[int], target: int) -> List[int]:
        seen = {}
        for i, num in enumerate(nums):
            complement = target - num
            if complement in seen:
                return [seen[complement], i]
            seen[num] = i
        return []
class Solution {
public:
    vector<int> twoSum(vector<int>& nums, int target) {
        unordered_map<int, int> seen; // key: gia tri, val: chi so
        for (int i = 0; i < nums.size(); i++) {
            int complement = target - nums[i];
            if (seen.count(complement)) {
                return {seen[complement], i}; // Tra ve cap index tim thay
            }
            seen[nums[i]] = i; // Ghi nhan vi tri phan tu
        }
        return {};
    }
};
