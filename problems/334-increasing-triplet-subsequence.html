Increasing Triplet Subsequence

O(n) O(1) 39.4% AC
Mô tả
Cho mảng nums. Kiểm tra có tồn tại bộ ba chỉ số i<j<k sao nums[i]<nums[j]<nums[k].
Ví dụ
Ví dụ 1
Innums = [1,2,3,4,5]
Outtrue
Ví dụ 2
Innums = [5,4,3,2,1]
Outfalse
Ví dụ 3
Innums = [2,1,5,0,4,6]
Outtrue
Ý tưởng
Greedy hai ngưỡng first, second nhỏ nhất: gặp x<first→first=x; x<first< x<second thì true; x<second cập nhật second.
Mã nguồn
solution.c C99 
bool increasingTriplet(int* nums, int n) {
    int first = INT_MAX, second = INT_MAX;
    for (int i = 0; i < n; i++) {
        if (nums[i] <= first) first = nums[i];
        else if (nums[i] <= second) second = nums[i];
        else return true;
    }
    return false;
}
class Solution:
    def increasingTriplet(self, nums: List[int]) -> bool:
        first = second = float('inf')
        for x in nums:
            if x <= first:
                first = x
            elif x <= second:
                second = x
            else:
                return True
        return False
