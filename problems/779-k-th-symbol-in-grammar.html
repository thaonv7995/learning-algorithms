K-th Symbol in Grammar

O(log k) O(log k) 48.3% AC
Mô tả
Hàng 1 của tam giác grammar: 0. Mỗi hàng: thay 0→01, 1→10. Trả k-th bit (1-indexed) ở hàng n.
Ví dụ
Ví dụ 1
Inn = 1, k = 1
Out0
Ví dụ 2
Inn = 2, k = 1
Out0
Ví dụ 3
Inn = 2, k = 2
Out1
Ý tưởng
Đệ quy: hàng n, k: parent = (k+1)/2 ở hàng n-1; nếu k lẻ giữ parent; chẵn thì flip parent bit.
Mã nguồn
solution.c C99 
int kthGrammar(int n, int k) {
    if(n==1) return 0;
    int p=kthGrammar(n-1,(k+1)/2);
    return k%2? p : 1-p;
}
class Solution:
    def kthGrammar(self, n: int, k: int) -> int:
        if n == 1: return 0
        parent = self.kthGrammar(n - 1, (k + 1) // 2)
        return parent if k % 2 else 1 - parent
class Solution {
public:
    int kthGrammar(int n, int k) {

        return k%2? p : 1-p;
    }
};
