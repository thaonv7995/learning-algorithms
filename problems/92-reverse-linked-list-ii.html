Reverse Linked List II

O(n) O(1) 51.5% AC
Mô tả
Cho linked list và left, right (1-indexed). Đảo ngược các nút từ vị trí left đến right inclusive, trả về head.
Ví dụ
Ví dụ 1
Inhead = [1,2,3,4,5], left = 2, right = 4
Out[1,4,3,2,5]
Ví dụ 2
Inhead = [5], left = 1, right = 1
Out[5]
Ý tưởng
Dummy; prev đến nút trước left. Đảo từng nút trong đoạn: cur=prev.next luôn; prev.next trỏ cur.next, cur.next nối sau prev.next cũ — giống reverse N nodes.
Mã nguồn
solution.c C99 
/* reverseBetween — dummy + local reverse; xem C++ */
class Solution:
    def reverseBetween(self, head: Optional[ListNode], left: int, right: int) -> Optional[ListNode]:
        dummy = ListNode(0, head); prev = dummy
        for _ in range(left - 1): prev = prev.next
        for _ in range(right - left):
            cur = prev.next
            prev.next = cur.next
            cur.next = prev.next.next
            prev.next.next = cur
        return dummy.next
class Solution {
public:
    ListNode* reverseBetween(ListNode* head, int left, int right) {
        ListNode dummy(0, head), *prev = &dummy;
        for (int i=0;i<left-1;i++) prev = prev->next;
        for (int i=0;i<right-left;i++) {
            ListNode* cur = prev->next;
            prev->next = cur->next;
            cur->next = prev->next->next;
            prev->next->next = cur;
        }
        return dummy.next;
    }
};
