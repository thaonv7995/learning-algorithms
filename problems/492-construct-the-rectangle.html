Construct the Rectangle

O(sqrt(area)) O(1) 63.8% AC
Mô tả
Diện tích area hình chữ nhật. Trả [length, width] với length≥width và lệch nhau tối thiểu (gần sqrt nhất).
Ví dụ
Ví dụ 1
Inarea = 4
Out[2,2]
Ví dụ 2
Inarea = 37
Out[37,1]
Ví dụ 3
Inarea = 122122
Out[427,286]
Ý tưởng
i từ floor(sqrt(area)) xuống 1; nếu area%i==0 return [area/i, i].
Mã nguồn
solution.c C99 
int* constructRectangle(int area, int* rs) {
    static int ans[2];
    int w=(int)sqrt(area);
    while(area%w) w--;
    ans[0]=area/w; ans[1]=w; *rs=2; return ans;
}
class Solution:
    def constructRectangle(self, area: int) -> List[int]:
        w = int(area ** 0.5)
        while area % w: w -= 1
        return [area // w, w]
class Solution {
public:
    vector<int> constructRectangle(int area) {
        int w=sqrt(area);
        while(area%w) w--;
        return {area/w, w};
    }
};
