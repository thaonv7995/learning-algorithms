Find Peak Element

O(log n) O(1) 47% AC
Mô tả
Cho mảng nums, nums[i] ≠ nums[i+1]. Peak là phần tử lớn hơn hàng xóm (nums[-1]=nums[n]=-∞). Trả index bất kỳ của một peak.
Ví dụ
Ví dụ 1
Innums = [1,2,3,1]
Out2
Ví dụ 2
Innums = [1,2,1,3,5,6,4]
Out5
Ý tưởng
Binary search: nếu nums[mid] < nums[mid+1] thì peak ở bên phải (lo=mid+1), ngược lại peak ở trái hoặc tại mid (hi=mid).
Mã nguồn
solution.c C99 
int findPeakElement(int* nums, int n) {
    int lo = 0, hi = n - 1;
    while (lo < hi) {
        int mid = lo + (hi - lo) / 2;
        if (nums[mid] < nums[mid + 1]) lo = mid + 1;
        else hi = mid;
    }
    return lo;
}
class Solution:
    def findPeakElement(self, nums: List[int]) -> int:
        lo, hi = 0, len(nums) - 1
        while lo < hi:
            mid = (lo + hi) // 2
            if nums[mid] < nums[mid + 1]:
                lo = mid + 1
            else:
                hi = mid
        return lo
