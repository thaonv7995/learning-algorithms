Decode Ways

O(n) O(1) 38% AC
Mô tả
Chuỗi s chỉ gồm chữ số. Đếm số cách giải mã: '1'→'A', …, '26'→'Z'. Không có số 0 đứng đầu mã hợp lệ (vd. '06' không hợp lệ).
Ví dụ
Ví dụ 1
Ins = "12"
Out2
Ví dụ 2
Ins = "226"
Out3
Ý tưởng
DP: ways[i] = cách decode prefix i ký tự. ways[i] += ways[i-1] nếu s[i-1]≠'0'; += ways[i-2] nếu s[i-2:i] trong 10..26. Rolling prev2, prev1.
Mã nguồn
solution.c C99 
/* numDecodings — DP rolling; xem C++ */
class Solution:
    def numDecodings(self, s: str) -> int:
        if not s or s[0] == '0': return 0
        p2, p1 = 1, 1
        for i in range(1, len(s)):
            cur = 0
            if s[i] != '0': cur += p1
            two = int(s[i-1:i+1])
            if 10 <= two <= 26: cur += p2
            p2, p1 = p1, cur
        return p1
class Solution {
public:
    int numDecodings(string s) {
        if (s.empty()||s[0]=='0') return 0;
        int p2=1,p1=1;
        for (int i=1;i<(int)s.size();i++) {
            int cur=0;
            if (s[i]!='0') cur+=p1;
            int two=(s[i-1]-'0')*10+(s[i]-'0');
            if (two>=10&&two<=26) cur+=p2;
            p2=p1; p1=cur;
        }
        return p1;
    }
};
Dynamic Programming O(n) O(1)
  1. 1Khởi tạo — DP: ways[i] = cách decode prefix i ký tự.
  2. 2Duyệt chính — ways[i] += ways[i-1] nếu s[i-1]≠'0'; += ways[i-2] nếu…
  3. 3Cập nhật — Rolling prev2, prev1.
