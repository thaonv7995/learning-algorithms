Decode Ways

O(N) O(1) 38% AC
Mô tả
Một thông điệp chứa các chữ cái từ A-Z được mã hóa thành các chữ số sử dụng ánh xạ: 'A' -> "1", 'B' -> "2"... 'Z' -> "26". Cho một chuỗi s chứa toàn ký tự số, hãy tính số lượng cách giải mã chuỗi đó thành thông điệp chữ cái hợp lệ.
Ví dụ
Ví dụ 1
Ins = "226"
Out3 (Có thể giải mã thành "BZ" (2 26), "VF" (22 6), hoặc "BBF" (2 2 6))
Ý tưởng
Áp dụng Quy hoạch động. Gọi `dp[i]` là số lượng cách giải mã chuỗi con `s[0...i-1]`. Tại mỗi ký tự `s[i-1]`, ta có hai khả năng giải mã:
- Giải mã đơn ký tự: hợp lệ nếu `s[i-1] != '0'`. Kết quả cộng thêm `dp[i-1]`.
- Giải mã song ký tự: ghép hai chữ số `s[i-2]` và `s[i-1]`, hợp lệ nếu giá trị ghép nằm trong khoảng `10 - 26`. Kết quả cộng thêm `dp[i-2]`.
Tối ưu hóa bộ nhớ xuống O(1) bằng cách chỉ sử dụng hai biến tích lũy thay vì mảng.
Mã nguồn
solution.c C99 
// Solution in C (Decode Ways)
int numDecodings(char* s) {
    int len = strlen(s);
    if (len == 0 || s[0] == '0') return 0;
    int prev2 = 1, prev1 = 1;
    for (int i = 1; i < len; i++) {
        int curr = 0;
        if (s[i] != '0') curr += prev1;
        int twoDigit = (s[i-1] - '0') * 10 + (s[i] - '0');
        if (twoDigit >= 10 && twoDigit <= 26) curr += prev2;
        prev2 = prev1;
        prev1 = curr;
    }
    return prev1;
}
# Solution in Python (Decode Ways)
class Solution:
    def numDecodings(self, s: str) -> int:
        if not s or s[0] == '0':
            return 0
        dp = [0] * (len(s) + 1)
        dp[0] = dp[1] = 1
        for i in range(2, len(s) + 1):
            if s[i - 1] != '0':
                dp[i] += dp[i - 1]
            two_digit = int(s[i - 2:i])
            if 10 <= two_digit <= 26:
                dp[i] += dp[i - 2]
        return dp[-1]
class Solution {
public:
    int numDecodings(string s) {
        if (s.empty() || s[0] == '0') return 0;
        int prev2 = 1; // tuong duong dp[i-2]
        int prev1 = 1; // tuong duong dp[i-1]
        for (int i = 1; i < s.length(); i++) {
            int current = 0;
            if (s[i] != '0') current += prev1; // Giai ma don
            int twoDigit = stoi(s.substr(i - 1, 2));
            if (twoDigit >= 10 && twoDigit <= 26) {
                current += prev2; // Giai ma kep
            }
            prev2 = prev1;
            prev1 = current;
        }
        return prev1;
    }
};
