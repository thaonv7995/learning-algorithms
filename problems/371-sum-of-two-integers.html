Sum of Two Integers

O(1) O(1) 55.5% AC
Mô tả
Cho hai số nguyên a và b. Tính tổng a+b không được dùng toán tử cộng (+) và trừ (-); chỉ dùng phép bit và logic.
Ví dụ
Ví dụ 1
Ina = 1, b = 2
Out3
Ví dụ 2
Ina = 2, b = 3
Out5
Ý tưởng
XOR cho sum không carry; AND<<1 cho carry; lặp đến carry=0. Xử lý số âm với mask 32-bit trong Python.
Mã nguồn
solution.c C99 
int getSum(int a, int b) {
    while (b) {
        int carry = (a & b) << 1;
        a = a ^ b;
        b = carry;
    }
    return a;
}
class Solution:
    def getSum(self, a: int, b: int) -> int:
        mask = 0xFFFFFFFF
        while b:
            a, b = (a ^ b) & mask, ((a & b) << 1) & mask
        return a if a <= 0x7FFFFFFF else ~(a ^ mask)
class Solution {
public:
    int getSum(int a, int b) {
        while (b) {
            unsigned carry = (unsigned)(a & b) << 1;
            a = a ^ b;
            b = (int)carry;
        }
        return a;
    }
};
