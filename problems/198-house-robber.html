House Robber

O(n) O(1) 53.2% AC
Mô tả
Dãy nhà nums[i] là tiền trong nhà i. Không được cướp hai nhà liền kề. Tìm số tiền tối đa có thể cướp.
Ví dụ
Ví dụ 1
Innums = [1,2,3,1]
Out4
Ví dụ 2
Innums = [2,7,9,3,1]
Out12
Ví dụ 3
Innums = [2,1,1,2]
Out4
Ý tưởng
DP: rob(i) = max(rob(i-1), rob(i-2)+nums[i]). Cuộn 2 biến prev2, prev1 — không cướp nhà i-1 hoặc cướp i cộng best đến i-2.
Mã nguồn
solution.c C99 
int rob(int* nums, int n) {
    int prev2 = 0, prev1 = 0;
    for (int i = 0; i < n; i++) {
        int cur = prev1 > prev2 + nums[i] ? prev1 : prev2 + nums[i];
        prev2 = prev1;
        prev1 = cur;
    }
    return prev1;
}
class Solution:
    def rob(self, nums: List[int]) -> int:
        prev2 = prev1 = 0
        for x in nums:
            prev2, prev1 = prev1, max(prev1, prev2 + x)
        return prev1
class Solution {
public:
    int rob(vector<int>& nums) {
        int prev2 = 0, prev1 = 0;
        for (int x : nums) {
            int cur = max(prev1, prev2 + x);
            prev2 = prev1;
            prev1 = cur;
        }
        return prev1;
    }
};
