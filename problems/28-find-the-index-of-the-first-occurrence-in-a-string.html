Find the Index of the First Occurrence in a String

O(M·N) O(1) 46.7% AC
Mô tả
Tìm index đầu tiên của needle trong haystack, hoặc -1 nếu không có.
Ví dụ
Ví dụ 1
Inhaystack = "sadbutsad", needle = "sad"
Out0
Ví dụ 2
Inhaystack = "leetcode", needle = "leeto"
Out-1
Ý tưởng
Với mỗi i trong haystack, so sánh haystack[i..i+len(needle)) với needle. Hoặc built-in find/KMP nâng cao.
Mã nguồn
solution.c C99 
int strStr(char* haystack, char* needle) {
    int n = strlen(haystack), m = strlen(needle);
    if (!m) return 0;
    for (int i = 0; i <= n - m; i++)
        if (!strncmp(haystack + i, needle, m)) return i;
    return -1;
}
class Solution:
    def strStr(self, haystack: str, needle: str) -> int:
        if not needle: return 0
        n, m = len(haystack), len(needle)
        for i in range(n - m + 1):
            if haystack[i:i+m] == needle: return i
        return -1
class Solution {
public:
    int strStr(string haystack, string needle) {
        if (needle.empty()) return 0;
        for (int i = 0; i + (int)needle.size() <= (int)haystack.size(); i++)
            if (haystack.compare(i, needle.size(), needle) == 0) return i;
        return -1;
    }
};
