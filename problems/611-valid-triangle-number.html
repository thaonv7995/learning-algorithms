Valid Triangle Number

O(n^2) O(1) 56.9% AC
Mô tả
Cho mảng nums. Đếm số bộ ba (i,j,k) i<j<k tạo tam giác (a+b>c).
Ví dụ
Ví dụ 1
Innums = [2,2,3,4]
Out3
Ví dụ 2
Innums = [4,2,3,4]
Out4
Ý tưởng
Sort nums; với mỗi k cố định (cạnh lớn nhất), two pointers l,r tìm cặp sao nums[l]+nums[r]>nums[k].
Mã nguồn
solution.c C99 
int triangleNumber(int* nums, int n) {
    qsort(nums,n,sizeof(int),cmp);
    int ans=0;
    for(int k=n-1;k>=2;k--){
        int l=0,r=k-1;
        while(l<r){
            if(nums[l]+nums[r]>nums[k]){ ans+=r-l; r--; }
            else l++;
        }
    }
    return ans;
}
class Solution:
    def triangleNumber(self, nums: List[int]) -> int:
        nums.sort(); n=len(nums); ans=0
        for k in range(n-1,1,-1):
            l,r=0,k-1
            while l<r:
                if nums[l]+nums[r]>nums[k]: ans+=r-l; r-=1
                else: l+=1
        return ans
