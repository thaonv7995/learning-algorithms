Reaching Points

O(log max) O(1) 34.4% AC
Mô tả
Điểm start (sx,sy) đến target (tx,ty) bằng thao tác: từ (x,y) có thể tới (x+y,y) hoặc (x,x+y). Có thể tới target không?
Ví dụ
Ví dụ 1
Insx = 1, sy = 1, tx = 3, ty = 5
Outtrue
Ví dụ 2
Insx = 1, sy = 1, tx = 2, ty = 2
Outfalse
Ví dụ 3
Insx = 1, sy = 1, tx = 1, ty = 1
Outtrue
Ý tưởng
Reverse từ target về start: while tx>=ty swap/mod; ty -= tx hoặc ty %= tx. Khả thi nếu về (sx,sy).
Mã nguồn
solution.c C99 
bool reachingPoints(int sx, int sy, int tx, int ty) {
    while(tx>=sx && ty>=sy){ if(tx==ty) break;
        if(tx>ty){ if(ty<sy) return false; if(tx>=ty*2) tx%=ty; else tx-=ty; }
        else { if(tx<sx) return false; if(ty>=tx*2) ty%=tx; else ty-=tx; }
    } return tx==sx && ty==sy;
}
class Solution:
    def reachingPoints(self, sx: int, sy: int, tx: int, ty: int) -> bool:
        while tx >= sx and ty >= sy:
            if tx == ty: break
            if tx > ty:
                if ty < sy: return False
                if tx >= ty * 2: tx %= ty
                else: tx -= ty
            else:
                if tx < sx: return False
                if ty >= tx * 2: ty %= tx
                else: ty -= tx
        return tx == sx and ty == sy
