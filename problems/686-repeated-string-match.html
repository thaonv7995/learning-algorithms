Repeated String Match

O(n*m) O(1) 38.9% AC
Mô tả
Cho A và B. A+B lặp lại tối thiểu bao nhiêu lần để B là substring?
Ví dụ
Ví dụ 1
InA="abcd", B="cdabcdab"
Out3
Ví dụ 2
InA="a", B="aa"
Out2
Ý tưởng
Try repeat A up to len(B)/len(A)+2; check substring; if fail len(A)+len(B).

class Solution:
    def repeatedStringMatch(self, a: str, b: str) -> int:
        n=(len(b)+len(a)-1)//len(a)
        for k in (n,n+1,n+2):
            if b in a*k: return k
        return -1
class Solution {
public:
    int repeatedStringMatch(string a, string b) {
        int n=(b.size()+a.size()-1)/a.size();
        string s;
        for(int k=n;k<=n+2;k++){ s=""; for(int i=0;i<k;i++) s+=a; if(s.find(b)!=string::npos) return k; }
        return -1;
    }
};
