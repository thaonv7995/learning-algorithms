Move Zeroes

O(N) O(1) 63.8% AC
Mô tả
Cho một mảng nums, hãy di chuyển toàn bộ các số 0 về cuối mảng trong khi vẫn giữ nguyên thứ tự tương đối của các phần tử khác 0 ban đầu.
Ví dụ
Ví dụ 1
Innums = [0,1,0,3,12]
Out[1,3,12,0,0]
Ý tưởng
Sử dụng hai con trỏ chạy trên cùng một mảng. Con trỏ `lastNonZero` dùng để ghi nhận vị trí điền số khác 0 tiếp theo. Khi duyệt mảng bằng con trỏ `i`, nếu gặp số khác 0, ta thực hiện hoán đổi (swap) giá trị giữa `nums[i]` và `nums[lastNonZero]`, sau đó tăng `lastNonZero` lên 1 đơn vị.
Mã nguồn
solution.c C99 
// Solution in C (Move Zeroes)
void moveZeroes(int* nums, int numsSize) {
    int lastNonZero = 0;
    for (int i = 0; i < numsSize; i++) {
        if (nums[i] != 0) {
            int tmp = nums[lastNonZero];
            nums[lastNonZero++] = nums[i];
            nums[i] = tmp;
        }
    }
}
# Solution in Python (Move Zeroes)
class Solution:
    def moveZeroes(self, nums: List[int]) -> None:
        last = 0
        for i in range(len(nums)):
            if nums[i] != 0:
                nums[last], nums[i] = nums[i], nums[last]
                last += 1
class Solution {
public:
    void moveZeroes(vector<int>& nums) {
        int lastNonZero = 0;
        for (int i = 0; i < nums.size(); i++) {
            if (nums[i] != 0) {
                swap(nums[lastNonZero++], nums[i]);
            }
        }
    }
};
