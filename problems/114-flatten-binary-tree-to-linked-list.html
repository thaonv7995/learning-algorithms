Flatten Binary Tree to Linked List

O(n) O(h) 70.7% AC
Mô tả
Biến cây nhị phân thành linked list theo thứ tự preorder (gốc → trái → phải), dùng con phải làm next. In-place, left = null.
Ví dụ
Ví dụ 1
Inroot = [1,2,5,3,4,null,6]
Out[1,null,2,null,3,null,4,null,5,null,6]
Ví dụ 2
Inroot = []
Out[]
Ý tưởng
DFS reverse postorder (phải → trái → gốc): gán node.right = prev, left = null, prev = node. Tương đương preorder khi nối ngược.
Mã nguồn
solution.c C99 
/* Reverse postorder nối prev — xem C++ */
class Solution:
    def flatten(self, root: Optional[TreeNode]) -> None:
        prev = None
        def dfs(node):
            nonlocal prev
            if not node: return
            dfs(node.right)
            dfs(node.left)
            node.right = prev
            node.left = None
            prev = node
        dfs(root)
class Solution {
    TreeNode* prev = nullptr;
    void dfs(TreeNode* nd) {
        if (!nd) return;
        dfs(nd->right);
        dfs(nd->left);
        nd->right = prev;
        nd->left = nullptr;
        prev = nd;
    }
public:
    void flatten(TreeNode* root) {
        dfs(root);
    }
};
