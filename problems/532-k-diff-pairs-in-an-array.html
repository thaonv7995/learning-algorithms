K-diff Pairs in an Array

O(n) O(n) 45.9% AC
Mô tả
Cho mảng nums và số k. Đếm số cặp (i,j) với i<j sao nums[i]-nums[j]=k hoặc nums[j]-nums[i]=k. Mỗi cặp giá trị chỉ đếm một lần.
Ví dụ
Ví dụ 1
Innums = [3,1,4,1,5], k = 2
Out2
Ví dụ 2
Innums = [1,2,3,4,5], k = 1
Out4
Ví dụ 3
Innums = [1,3,1,5,4], k = 0
Out1
Ý tưởng
HashMap đếm tần suất. Với mỗi x, cộng count[x+k] + (k==0 ? count[x]-1 : count[x-k]?) — chuẩn: duyệt x, cộng seen[x-k], rồi tăng seen[x].
Mã nguồn
solution.c C99 
/* Hash freq x-k — xem C++ */
class Solution:
    def findPairs(self, nums: List[int], k: int) -> int:
        from collections import Counter
        cnt = Counter()
        ans = 0
        for x in nums:
            if k == 0:
                if cnt[x]:
                    ans += 1
            elif cnt[x - k]:
                ans += 1
            cnt[x] += 1
        return ans
class Solution {
public:
    int findPairs(vector<int>& nums, int k) {
        unordered_map<int,int> cnt;
        int ans = 0;
        for (int x : nums) {
            if (k == 0) { if (cnt[x]) ans++; }
            else if (cnt.count(x - k)) ans++;
            cnt[x]++;
        }
        return ans;
    }
};
