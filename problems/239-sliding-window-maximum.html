Sliding Window Maximum

O(N) (Mỗi phần tử được đưa vào/rút ra Deque tối đa 1 lần) O(K) (Kích thước tối đa của Deque) 48.8% AC
Mô tả
Cho một mảng số nguyên nums và kích thước cửa sổ trượt k liên tục dịch từ trái sang phải mỗi lần 1 phần tử. Hãy trả về một mảng chứa giá trị lớn nhất trong mỗi cửa sổ trượt.
Ví dụ
Ví dụ 1
Innums = [1,3,-1,-3,5,3,6,7], k = 3
Out[3,3,5,5,6,7]
Ý tưởng
Sử dụng Hàng đợi hai đầu đơn điệu (Monotonic Deque) lưu trữ các chỉ số (index) của mảng. Deque luôn duy trì các giá trị tương ứng với chỉ số giảm dần từ đầu đến cuối Deque. Mỗi khi duyệt qua phần tử `nums[i]`:
1. Loại bỏ các chỉ số đã trượt ra ngoài cửa sổ bên trái khỏi Deque (`index < i - k + 1`).
2. Loại bỏ tất cả các chỉ số ở cuối Deque có giá trị tương ứng nhỏ hơn `nums[i]` (vì chúng không còn cơ hội làm giá trị lớn nhất).
3. Đẩy chỉ số `i` vào cuối Deque.
4. Đỉnh Deque (đầu hàng) chính là index chứa giá trị lớn nhất của cửa sổ hiện tại.
Mã nguồn
solution.c C99 
// Solution in C (Sliding Window Maximum)
// Requires Monotonic Deque layout representation.
# Solution in Python (Sliding Window Maximum)
from collections import deque
class Solution:
    def maxSlidingWindow(self, nums: List[int], k: int) -> List[int]:
        res = []
        q = deque() # index stores
        for i, num in enumerate(nums):
            while q and nums[q[-1]] < num:
                q.pop()
            q.append(i)
            if q[0] == i - k:
                q.popleft()
            if i >= k - 1:
                res.append(nums[q[0]])
        return res
class Solution {
public:
    vector<int> maxSlidingWindow(vector<int>& nums, int k) {
        vector<int> res;
        deque<int> dq; // Luu tru index phan tu
        for (int i = 0; i < nums.size(); i++) {
            if (!dq.empty() && dq.front() < i - k + 1) {
                dq.pop_front(); // Loai bo index ra ngoai cua so truyen
            }
            while (!dq.empty() && nums[dq.back()] < nums[i]) {
                dq.pop_back(); // Duy tri tinh don dieu giam dan
            }
            dq.push_back(i);
            if (i >= k - 1) res.push_back(nums[dq.front()]);
        }
        return res;
    }
};
