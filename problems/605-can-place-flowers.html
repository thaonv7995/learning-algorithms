Can Place Flowers

O(n) O(1) 29.1% AC
Mô tả
Cho mảng flowerbed (0=trống, 1=có hoa) và n. Có thể trồng thêm n cây sao không hai cây liền kề? Trả true/false.
Ví dụ
Ví dụ 1
Inflowerbed = [1,0,0,0,1], n = 1
Outtrue
Ví dụ 2
Inflowerbed = [1,0,0,0,1], n = 2
Outfalse
Ví dụ 3
Inflowerbed = [0,0,1,0,0], n = 1
Outtrue
Ý tưởng
Greedy quét trái→phải: nếu flowerbed[i]==0 và hai bên trống thì trồng (đặt 1) và n--.
Mã nguồn
solution.c C99 
bool canPlaceFlowers(int* f, int m, int n) {
    for(int i=0;i<m;i++){
        int l=i?f[i-1]:0, r=i+1<m?f[i+1]:0;
        if(!f[i]&&!l&&!r){ f[i]=1; if(--n<=0) return 1; }
    }
    return n<=0;
}
class Solution:
    def canPlaceFlowers(self, flowerbed: List[int], n: int) -> bool:
        bed = [0] + flowerbed + [0]
        for i in range(1, len(bed)-1):
            if bed[i-1]==0 and bed[i]==0 and bed[i+1]==0:
                bed[i]=1; n-=1
                if n<=0: return True
        return n<=0
class Solution {
public:
    bool canPlaceFlowers(vector<int>& f, int n) {
        int m=f.size();
        for(int i=0;i<m;i++){
            int l=i?f[i-1]:0, r=i+1<m?f[i+1]:0;
            if(!f[i]&&!l&&!r){ f[i]=1; if(--n<=0) return true; }
        }
        return n<=0;
    }
};
