Merge Sorted Array

O(m+n) O(1) 54.9% AC
Mô tả
Hai mảng sort nums1 (length m+n, m phần tử đầu hợp lệ) và nums2 (length n). Gộp nums2 vào nums1 in-place, kết quả sort tăng dần.
Ví dụ
Ví dụ 1
Innums1 = [1,2,3,0,0,0], m = 3, nums2 = [2,5,6], n = 3
Out[1,2,2,3,5,6]
Ví dụ 2
Innums1 = [1], m = 1, nums2 = [], n = 0
Out[1]
Ý tưởng
Merge từ cuối: con trỏ i=m-1, j=n-1, k=m+n-1; chọn lớn hơn ghi vào nums1[k], tránh ghi đè phần chưa merge của nums1.

void merge(int* nums1, int m, int* nums2, int n) {
    int i=m-1, j=n-1, k=m+n-1;
    while (j>=0) {
        if (i>=0 && nums1[i]>nums2[j]) nums1[k--]=nums1[i--];
        else nums1[k--]=nums2[j--];
    }
}
class Solution:
    def merge(self, nums1: List[int], m: int, nums2: List[int], n: int) -> None:
        i, j, k = m - 1, n - 1, m + n - 1
        while j >= 0:
            if i >= 0 and nums1[i] > nums2[j]:
                nums1[k] = nums1[i]; i -= 1
            else:
                nums1[k] = nums2[j]; j -= 1
            k -= 1
class Solution {
public:
    void merge(vector<int>& a, int m, vector<int>& b, int n) {
        int i=m-1, j=n-1, k=m+n-1;
        while (j>=0) {
            if (i>=0 && a[i]>b[j]) a[k--]=a[i--];
            else a[k--]=b[j--];
        }
    }
};
