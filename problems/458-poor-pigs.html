Poor Pigs

O(1) O(1) 59.1% AC
Mô tả
Có buckets và pigs để test poison trong một bucket. Mỗi pig uống mix buckets, 15 phút sau biết sống/chết. Min pigs để xác định poison bucket trong t minutes.
Ví dụ
Ví dụ 1
Inbuckets = 1000, minutesToTest = 15, minutesToDie = 60
Out5
Ví dụ 2
Inbuckets = 1, minutesToTest = 1, minutesToDie = 1
Out0
Ý tưởng
Mỗi pig có states = rounds+1 (sống/chết mỗi round). Tổng states^pigs ≥ buckets → min pigs.
Mã nguồn
solution.c C99 
int poorPigs(int buckets, int minutesToTest, int minutesToDie) {
    int states=minutesToDie/minutesToTest+1, pigs=0, cap=1;
    while(cap<buckets){ pigs++; cap*=states; }
    return pigs;
}
class Solution:
    def poorPigs(self, buckets: int, minutesToTest: int, minutesToDie: int) -> int:
        states = minutesToDie // minutesToTest + 1
        pigs = 0
        while states ** pigs < buckets:
            pigs += 1
        return pigs
