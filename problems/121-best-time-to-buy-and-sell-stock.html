Best Time to Buy and Sell Stock

O(n) O(1) 56.8% AC
Mô tả
Cho mảng giá cổ phiếu theo ngày (prices[i] là giá ngày i). Chỉ được mua tối đa một lần và bán tối đa một lần (phải mua trước khi bán). Tìm lợi nhuận tối đa.
Ví dụ
Ví dụ 1
Inprices = [7,1,5,3,6,4]
Out5
Ví dụ 2
Inprices = [7,6,4,3,1]
Out0
Ý tưởng
Một lần duyệt: giữ minPrice (giá thấp nhất đã thấy). Tại mỗi ngày, lợi nhuận = price - minPrice; cập nhật maxProfit và minPrice.
Mã nguồn
solution.c C99 
/* Một pass min price + max profit — xem C++ */
class Solution:
    def maxProfit(self, prices: List[int]) -> int:
        min_p, best = float('inf'), 0
        for p in prices:
            min_p = min(min_p, p)
            best = max(best, p - min_p)
        return best
class Solution {
public:
    int maxProfit(vector<int>& prices) {
        int minP = INT_MAX, best = 0;
        for (int p : prices) {
            minP = min(minP, p);
            best = max(best, p - minP);
        }
        return best;
    }
};
