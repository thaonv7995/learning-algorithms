Rabbits in Forest

O(n) O(n) 58.1% AC
Mô tả
Mỗi con thỏ trả lời có bao nhiêu con thỏ khác cùng màu. answers[i] là câu trả lời của thỏ i. Có thể có bao nhiêu con thỏ trong rừng tối đa?
Ví dụ
Ví dụ 1
Inanswers = [1,1,2]
Out5
Ví dụ 2
Inanswers = [1,0]
Out2
Ví dụ 3
Inanswers = [1,1,1,1,1]
Out0
Ý tưởng
Gom answers theo nhóm: mỗi nhóm k cùng answer x (x>0) cần x+1 thỏ (1 khác màu + x cùng màu). answer=0 là thỏ unique.
Mã nguồn
solution.c C99 
int numRabbits(int* answers, int n) {
    int cnt[1001]={0}; for(int i=0;i<n;i++) cnt[answers[i]]++;
    int ans=0; for(int x=0;x<1001;x++) if(cnt[x]){ int g=x+1; ans+=((cnt[x]+g-1)/g)*g; } return ans;
}
class Solution:
    def numRabbits(self, answers: List[int]) -> int:
        cnt = Counter(answers)
        ans = 0
        for x, c in cnt.items():
            g = x + 1
            ans += ((c + g - 1) // g) * g
        return ans
class Solution {
public:
    int numRabbits(vector<int>& answers) {
        unordered_map<int,int> cnt;
        for(int x:answers) cnt[x]++;
        int ans=0;
        for(auto& [x,c]: cnt){
            int g=x+1;
            ans += ((c+g-1)/g)*g;
        }
        return ans;
    }
};
