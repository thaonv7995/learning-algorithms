Biggest Single Number

O(n) O(n) 71.3% AC
Mô tả
Bảng MyNumbers(num). Trả số xuất hiện đúng một lần lớn nhất; nếu không có trả NULL.
Ví dụ
Ví dụ 1
Innum=8 xuất hiện 1 lần, num=3 hai lần
Out8
Ví dụ 2
InMọi số ≥2 lần
OutNULL
Ý tưởng
SELECT MAX(num) FROM MyNumbers GROUP BY num HAVING COUNT(*)=1 hoặc subquery.
Mã nguồn
solution.c C99 
const char query[] = "SELECT MAX(num) AS num\nFROM MyNumbers\nWHERE num IN (SELECT num FROM MyNumbers GROUP BY num HAVING COUNT(*) = 1);";
# SQL
SELECT MAX(num) AS num
FROM MyNumbers
WHERE num IN (SELECT num FROM MyNumbers GROUP BY num HAVING COUNT(*) = 1);
// SQL
SELECT MAX(num) AS num
FROM MyNumbers
WHERE num IN (SELECT num FROM MyNumbers GROUP BY num HAVING COUNT(*) = 1);
Database O(n) O(n)
