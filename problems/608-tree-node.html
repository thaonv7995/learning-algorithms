Tree Node

O(n) O(n) 75.7% AC
Mô tả
Bảng Tree(id, p_id) mô tả cây nhị phân (p_id NULL là root). Trả id và type: Root (p_id NULL), Leaf (id không là p_id của ai), Inner (còn lại).
Ví dụ
Ví dụ 1
In1 root, 2 và 3 con của 1
Out1 Root, 2 Leaf, 3 Leaf
Ví dụ 2
In1→2→3 chain
Out1 Root, 2 Inner, 3 Leaf
Ví dụ 3
InMột nút root duy nhất
Outtype=Root
Ý tưởng
CASE: p_id IS NULL → Root; id NOT IN (SELECT p_id FROM Tree WHERE p_id IS NOT NULL) → Leaf; else Inner.
Mã nguồn
solution.c C99 
const char query[] = "SELECT id, CASE WHEN p_id IS NULL THEN 'Root' WHEN id NOT IN (SELECT p_id FROM Tree WHERE p_id IS NOT NULL) THEN 'Leaf' ELSE 'Inner' END AS type FROM Tree;";
# SQL
SELECT id,
  CASE
    WHEN p_id IS NULL THEN 'Root'
    WHEN id NOT IN (SELECT p_id FROM Tree WHERE p_id IS NOT NULL) THEN 'Leaf'
    ELSE 'Inner'
  END AS type
FROM Tree;
