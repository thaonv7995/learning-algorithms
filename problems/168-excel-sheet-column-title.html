Excel Sheet Column Title

O(log n) O(1) 46.5% AC
Mô tả
Cho số cột columnNumber (1 → A, 2 → B, …, 26 → Z, 27 → AA). Trả tên cột Excel tương ứng.
Ví dụ
Ví dụ 1
IncolumnNumber = 1
Out"A"
Ví dụ 2
IncolumnNumber = 28
Out"AB"
Ví dụ 3
IncolumnNumber = 701
Out"ZY"
Ý tưởng
Biến đổi số sang base-26 nhưng không phải hệ thập phân thuần: mỗi bước columnNumber-- rồi lấy columnNumber%26, append 'A'+rem, columnNumber/=26.
Mã nguồn
solution.c C99 
char* convertToTitle(int n) {
    char tmp[32]; int k = 0;
    while (n) {
        n--;
        tmp[k++] = 'A' + (n % 26);
        n /= 26;
    }
    char* res = malloc(k + 1);
    for (int i = 0; i < k; i++) res[i] = tmp[k - 1 - i];
    res[k] = 0; return res;
}
class Solution:
    def convertToTitle(self, n: int) -> str:
        res = []
        while n:
            n -= 1
            res.append(chr(ord('A') + n % 26))
            n //= 26
        return ''.join(reversed(res))
class Solution {
public:
    string convertToTitle(int n) {
        string res;
        while (n) {
            n--;
            res += char('A' + n % 26);
            n /= 26;
        }
        reverse(res.begin(), res.end());
        return res;
    }
};
