Find Minimum in Rotated Sorted Array

O(log n) O(1) 54.7% AC
Mô tả
Cho mảng nums đã sort tăng dần được xoay vòng (pivot), không trùng lặp. Tìm phần tử nhỏ nhất của mảng trong O(log n).
Ví dụ
Ví dụ 1
Innums = [3,4,5,1,2]
Out1
Ví dụ 2
Innums = [4,5,6,7,0,1,2]
Out0
Ví dụ 3
Innums = [11,13,15,17]
Out11
Ý tưởng
Binary search: so nums[mid] với nums[hi]. Nếu nums[mid] > nums[hi] thì min nằm bên phải (lo = mid+1); ngược lại min ở trái hoặc tại mid (hi = mid). Kết thúc trả nums[lo].
Mã nguồn
solution.c C99 
int findMin(int* nums, int n) {
    int lo = 0, hi = n - 1;
    while (lo < hi) {
        int mid = lo + (hi - lo) / 2;
        if (nums[mid] > nums[hi]) lo = mid + 1;
        else hi = mid;
    }
    return nums[lo];
}
class Solution:
    def findMin(self, nums: List[int]) -> int:
        lo, hi = 0, len(nums) - 1
        while lo < hi:
            mid = (lo + hi) // 2
            if nums[mid] > nums[hi]:
                lo = mid + 1
            else:
                hi = mid
        return nums[lo]
