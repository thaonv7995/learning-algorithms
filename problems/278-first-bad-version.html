First Bad Version

O(log n) O(1) 47.1% AC
Mô tả
Có n phiên bản [1..n], phiên bản bad trở đi đều bad. API isBadVersion(v). Tìm bad version đầu tiên tối thiểu số lần gọi API.
Ví dụ
Ví dụ 1
Inn = 5, bad = 4
Out4
Ví dụ 2
Inn = 1, bad = 1
Out1
Ý tưởng
Binary search: mid; nếu isBadVersion(mid) thì bad ở mid hoặc trái → hi=mid; else lo=mid+1.
Mã nguồn
solution.c C99 
int firstBadVersion(int n) {
    int lo = 1, hi = n;
    while (lo < hi) {
        int mid = lo + (hi - lo) / 2;
        if (isBadVersion(mid)) hi = mid;
        else lo = mid + 1;
    }
    return lo;
}
class Solution:
    def firstBadVersion(self, n: int) -> int:
        lo, hi = 1, n
        while lo < hi:
            mid = (lo + hi) // 2
            if isBadVersion(mid):
                hi = mid
            else:
                lo = mid + 1
        return lo
