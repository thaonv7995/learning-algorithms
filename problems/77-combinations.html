Combinations

O(C(n,k)·k) O(k) 74.6% AC
Mô tả
Cho n và k. Trả về mọi tổ hợp gồm k số phân biệt chọn từ 1..n, mỗi tổ hợp tăng dần, thứ tự tổ hợp trong kết quả không quan trọng.
Ví dụ
Ví dụ 1
Inn = 4, k = 2
Out[[1,2],[1,3],[1,4],[2,3],[2,4],[3,4]]
Ví dụ 2
Inn = 1, k = 1
Out[[1]]
Ý tưởng
Backtracking: bắt đầu từ start, thêm i vào path, đệ quy i+1; khi len(path)==k thì push. Cắt nhánh khi còn không đủ số.
Mã nguồn
solution.c C99 
/* combine — backtracking; xem C++ */
class Solution:
    def combine(self, n: int, k: int) -> List[List[int]]:
        res, path = [], []
        def dfs(start):
            if len(path) == k:
                res.append(path[:]); return
            need = k - len(path)
            for i in range(start, n - need + 2):
                path.append(i); dfs(i + 1); path.pop()
        dfs(1)
        return res
class Solution {
    vector<vector<int>> res;
    vector<int> path;
    void dfs(int n, int k, int start) {
        if ((int)path.size()==k) { res.push_back(path); return; }
        int need = k - (int)path.size();
        for (int i=start; i<=n-need+1; i++) {
            path.push_back(i); dfs(n,k,i+1); path.pop_back();
        }
    }
public:
    vector<vector<int>> combine(int n, int k) {
        dfs(n,k,1); return res;
    }
};
