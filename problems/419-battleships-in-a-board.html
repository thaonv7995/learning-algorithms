Battleships in a Board

O(mn) O(1) 77.5% AC
Mô tả
Bảng 'X' (tàu) và '.' (nước). Đếm số battleship — mỗi tàu là một khối 'X' liên thông ngang/dọc, không chéo.
Ví dụ
Ví dụ 1
Inboard = [["X",".",".","X"],[".",".",".","X"],[".",".",".","X"]]
Out2
Ví dụ 2
Inboard = [["."]]
Out0
Ví dụ 3
Inboard = [["X"]]
Out1
Ý tưởng
Đếm 'X' mà ô trên hoặc trái không phải 'X' — mỗi ship chỉ có một góc trên-trái.

int countBattleships(char** board, int m, int* cs, int n) {
    int ans=0;
    for(int i=0;i<m;i++) for(int j=0;j<n;j++){
        if(board[i][j]!='X') continue;
        if(i>0&&board[i-1][j]=='X') continue;
        if(j>0&&board[i][j-1]=='X') continue;
        ans++;
    }
    return ans;
}
class Solution:
    def countBattleships(self, board: List[List[str]]) -> int:
        m, n = len(board), len(board[0])
        ans = 0
        for i in range(m):
            for j in range(n):
                if board[i][j] != "X": continue
                if i > 0 and board[i-1][j] == "X": continue
                if j > 0 and board[i][j-1] == "X": continue
                ans += 1
        return ans
