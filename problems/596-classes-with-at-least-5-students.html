Classes With at Least 5 Students

O(n) O(n) 63.8% AC
Mô tả
Bảng Courses(student, class). Trả các class có ≥5 học sinh (student có thể trùng class).
Ví dụ
Ví dụ 1
Inclass A có 5 student
OutA
Ví dụ 2
Inclass B có 4 student
OutKhông liệt kê B
Ý tưởng
SELECT class FROM Courses GROUP BY class HAVING COUNT(DISTINCT student) >= 5 (hoặc COUNT(*) nếu mỗi row unique).
Mã nguồn
solution.c C99 
/* SQL GROUP BY HAVING — xem SQL */
# SQL
SELECT class
FROM Courses
GROUP BY class
HAVING COUNT(*) >= 5;
// SQL
SELECT class
FROM Courses
GROUP BY class
HAVING COUNT(*) >= 5;
