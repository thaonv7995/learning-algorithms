Robot Return to Origin

O(n) O(1) 78.1% AC
Mô tả
Robot bắt đầu (0,0) hướng lên. Chuỗi moves UDLR — trả true nếu quay về origin.
Ví dụ
Ví dụ 1
Inmoves = "UD"
Outtrue
Ví dụ 2
Inmoves = "LL"
Outfalse
Ý tưởng
Đếm U-D và L-R; cả hai bằng 0 thì về origin.
Mã nguồn
solution.c C99 
bool judgeCircle(char* moves){ int x=0,y=0; for(;*moves;moves++){ if(*moves=='U')y++; else if(*moves=='D')y--; else if(*moves=='L')x--; else x++; } return x==0&&y==0; }
class Solution:
    def judgeCircle(self, moves: str) -> bool:
        x=y=0
        for m in moves:
            if m=="U": y+=1
            elif m=="D": y-=1
            elif m=="L": x-=1
            else: x+=1
        return x==0 and y==0
class Solution {
public:
    bool judgeCircle(string moves) {
        int x=0,y=0;
        for(char m:moves){
            if(m=='U') y++; else if(m=='D') y--;
            else if(m=='L') x--; else x++;
        }
        return !x&&!y;
    }
};
