Valid Parentheses

O(N) O(N) 44.2% AC
Mô tả
Cho một chuỗi s chỉ chứa các ký tự mở/đóng ngoặc '(', ')', '{', '}', '[' và ']', hãy kiểm tra xem chuỗi đầu vào có hợp lệ hay không.

Chuỗi hợp lệ phải mở ngoặc đúng loại và đóng ngoặc theo đúng thứ tự LIFO.
Ví dụ
Ví dụ 1
Ins = "()[]{}"
Outtrue
Ví dụ 2
Ins = "(]"
Outfalse
Ý tưởng
Sử dụng cấu trúc dữ liệu Ngăn xếp (Stack) để quản lý thứ tự lồng nhau của các dấu ngoặc. Khi gặp một ký tự mở '(', '{', '[', ta đẩy (push) nó vào Stack. Khi gặp một ký tự đóng, ta kiểm tra xem Stack có rỗng hay không; nếu rỗng hoặc ký tự đầu Stack không khớp loại mở tương ứng, chuỗi không hợp lệ. Khi kết thúc chuỗi, Stack phải trống hoàn toàn.

class Solution {
public:
    bool isValid(string s) {
        stack<char> st;
        for (char c : s) {
            if (c == '(' || c == '{' || c == '[') {
                st.push(c);
            } else {
                if (st.empty()) return false;
                char top = st.top();
                if ((c == ')' && top != '(') ||
                    (c == '}' && top != '{') ||
                    (c == ']' && top != '[')) return false;
                st.pop();
            }
        }
        return st.empty();
    }
};
