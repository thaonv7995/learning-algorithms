Remove Element

O(N) O(1) 61.8% AC
Mô tả
Xóa mọi phần tử bằng val in-place, trả về số phần tử còn lại k.
Ví dụ
Ví dụ 1
Innums = [3,2,2,3], val = 3
Out2, nums = [2,2,_,_]
Ví dụ 2
Innums = [0,1,2,2,3,0,4,2], val = 2
Out5
Ý tưởng
write=0, duyệt read: nếu nums[read]!=val thì nums[write++]=nums[read].
Mã nguồn
solution.c C99 
int removeElement(int* nums, int n, int val) {
    int w = 0;
    for (int i = 0; i < n; i++) if (nums[i] != val) nums[w++] = nums[i];
    return w;
}
class Solution:
    def removeElement(self, nums: List[int], val: int) -> int:
        w = 0
        for x in nums:
            if x != val:
                nums[w] = x; w += 1
        return w
class Solution {
public:
    int removeElement(vector<int>& nums, int val) {
        int w = 0;
        for (int x : nums) if (x != val) nums[w++] = x;
        return w;
    }
};
