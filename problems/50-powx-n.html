Pow(x, n)

O(log N) O(1) 38.7% AC
Mô tả
Tính x^n (n có thể âm). Không dùng hàm pow có sẵn.
Ví dụ
Ví dụ 1
Inx = 2.0, n = 10
Out1024.0
Ví dụ 2
Inx = 2.0, n = -2
Out0.25
Ý tưởng
Binary exponentiation: n chẵn → (x²)^(n/2); n lẻ → x * x^(n-1). n<0 → đảo x, n=-n.
Mã nguồn
solution.c C99 
double myPow(double x, int n) {
    long N = n;
    if (N < 0) { x = 1.0 / x; N = -N; }
    double res = 1.0;
    while (N) {
        if (N & 1) res *= x;
        x *= x; N >>= 1;
    }
    return res;
}
class Solution:
    def myPow(self, x: float, n: int) -> float:
        if n < 0: x, n = 1/x, -n
        res = 1.0
        while n:
            if n & 1: res *= x
            x *= x; n >>= 1
        return res
