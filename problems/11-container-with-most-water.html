Container With Most Water

O(n) O(1) 60% AC
Mô tả
Cho mảng height — chiều cao các cột tại index i. Chọn hai cột tạo thùng chứa (không nghiêng) có diện tích nước lớn nhất. Diện tích = min(h[l], h[r]) × (r − l).
Ví dụ
Ví dụ 1
Inheight = [1,8,6,2,5,4,8,3,7]
Out49
Ví dụ 2
Inheight = [1,1]
Out1
Ý tưởng
Two pointers left=0, right=n-1. Area = min(h[l],h[r])×(r-l). Di chuyển con trỏ ở cột thấp hơn vào trong (cột cao giữ lại).
Mã nguồn
solution.c C99 
/* Two pointers — xem C++ */
class Solution:
    def maxArea(self, height: List[int]) -> int:
        l, r, best = 0, len(height)-1, 0
        while l < r:
            best = max(best, min(height[l], height[r]) * (r - l))
            if height[l] < height[r]: l += 1
            else: r -= 1
        return best
class Solution {
public:
    int maxArea(vector<int>& height) {
        int l = 0, r = height.size()-1, best = 0;
        while (l < r) {
            best = max(best, min(height[l], height[r]) * (r - l));
            if (height[l] < height[r]) l++; else r--;
        }
        return best;
    }
};
