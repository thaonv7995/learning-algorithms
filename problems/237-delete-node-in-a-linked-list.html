Delete Node in a Linked List

O(1) O(1) 83.9% AC
Mô tả
Cho node cần xóa trong singly-linked list (không cho head, node chắc chắn không phải tail). Xóa node đó — không trả về gì.
Ví dụ
Ví dụ 1
Inhead = [4,5,1,9], node = 5
Out[4,1,9]
Ví dụ 2
Inhead = [4,5,1,9], node = 1
Out[4,5,9]
Ý tưởng
Copy val từ node.next vào node, rồi node.next = node.next.next — 'xóa' next thay vì node hiện tại.
Mã nguồn
solution.c C99 
void deleteNode(struct ListNode* node) {
    node->val = node->next->val;
    node->next = node->next->next;
}
class Solution:
    def deleteNode(self, node: ListNode) -> None:
        node.val = node.next.val
        node.next = node.next.next
class Solution {
public:
    void deleteNode(ListNode* node) {
        node->val = node->next->val;
        node->next = node->next->next;
    }
};
