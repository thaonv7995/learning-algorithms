Binary Tree Inorder Traversal

O(n) O(h) 80.1% AC
Mô tả
Cho root cây nhị phân. Trả về thứ tự duyệt inorder (trái → gốc → phải) — danh sách giá trị các nút.
Ví dụ
Ví dụ 1
Inroot = [1,null,2,3]
Out[1,3,2]
Ví dụ 2
Inroot = []
Out[]
Ý tưởng
Đệ quy: inorder(left), append root.val, inorder(right). Hoặc iterative: stack — luôn đẩy left chain, pop visit, sang right.
Mã nguồn
solution.c C99 
/* inorderTraversal — stack iterative; xem C++ */
class Solution:
    def inorderTraversal(self, root: Optional[TreeNode]) -> List[int]:
        res, st = [], []
        cur = root
        while cur or st:
            while cur:
                st.append(cur); cur = cur.left
            cur = st.pop(); res.append(cur.val); cur = cur.right
        return res
class Solution {
public:
    vector<int> inorderTraversal(TreeNode* root) {
        vector<int> res; stack<TreeNode*> st;
        TreeNode* cur = root;
        while (cur || !st.empty()) {
            while (cur) { st.push(cur); cur = cur->left; }
            cur = st.top(); st.pop(); res.push_back(cur->val);
            cur = cur->right;
        }
        return res;
    }
};
