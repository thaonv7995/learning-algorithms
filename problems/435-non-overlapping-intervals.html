Non-overlapping Intervals

O(n log n) O(1) 57.1% AC
Mô tả
Cho intervals[i]=[start,end]. Xóa tối thiểu interval để phần còn lại không overlap.
Ví dụ
Ví dụ 1
Inintervals = [[1,2],[2,3],[3,4],[1,3]]
Out1
Ví dụ 2
Inintervals = [[1,2],[1,2],[1,2]]
Out2
Ví dụ 3
Inintervals = [[1,2],[2,3]]
Out0
Ý tưởng
Sort theo end tăng; greedy giữ interval, bỏ interval tiếp theo nếu start < end đang giữ.
Mã nguồn
solution.c C99 
int cmp(const void* a, const void* b) { return (*(int**)a)[1] - (*(int**)b)[1]; }
int eraseOverlapIntervals(int** iv, int n, int* cs) {
    qsort(iv, n, sizeof(int*), cmp);
    int end=INT_MIN, rem=0;
    for(int i=0;i<n;i++) if(iv[i][0]>=end) end=iv[i][1]; else rem++;
    return rem;
}
class Solution:
    def eraseOverlapIntervals(self, intervals: List[List[int]]) -> int:
        intervals.sort(key=lambda x: x[1])
        end, removed = float("-inf"), 0
        for s, e in intervals:
            if s >= end: end = e
            else: removed += 1
        return removed
class Solution {
public:
    int eraseOverlapIntervals(vector<vector<int>>& v) {
        sort(v.begin(), v.end(), [](auto& a, auto& b){ return a[1] < b[1]; });
        int end=INT_MIN, rem=0;
        for(auto& p:v) if(p[0]>=end) end=p[1]; else rem++;
        return rem;
    }
};
