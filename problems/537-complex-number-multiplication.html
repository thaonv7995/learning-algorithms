Complex Number Multiplication

O(1) O(1) 73.5% AC
Mô tả
Cho hai số phức dạng chuỗi "a+bi". Trả tích dạng "c+di" (a,b,c,d nguyên).
Ví dụ
Ví dụ 1
Innum1 = "1+1i", num2 = "1+1i"
Out"0+2i"
Ví dụ 2
Innum1 = "1+-1i", num2 = "1+-1i"
Out"0+-2i"
Ý tưởng
Parse a,b từ num1 và c,d từ num2. (a+bi)(c+di) = (ac-bd) + (ad+bc)i.
Mã nguồn
solution.c C99 
/* Parse + multiply formula — xem C++ */
class Solution:
    def complexNumberMultiply(self, num1: str, num2: str) -> str:
        def parse(s):
            i = s.index('+')
            a = int(s[:i])
            b = int(s[i+1:-1])
            return a, b
        a, b = parse(num1)
        c, d = parse(num2)
        return f"{a*c-b*d}+{a*d+b*c}i"
class Solution {
    pair<int,int> parse(const string& s) {
        int p = s.find('+');
        int a = stoi(s.substr(0, p));
        int b = stoi(s.substr(p+1, s.size()-p-2));
        return {a,b};
    }
public:
    string complexNumberMultiply(string num1, string num2) {
        auto [a,b] = parse(num1);
        auto [c,d] = parse(num2);
        return to_string(a*c-b*d) + "+" + to_string(a*d+b*c) + "i";
    }
};
