Single Number II

O(n) O(1) 67.1% AC
Mô tả
Cho mảng nums: mọi phần tử xuất hiện đúng ba lần, trừ một phần tử xuất hiện một lần. Tìm phần tử đó. O(n) thời gian, O(1) bộ nhớ phụ.
Ví dụ
Ví dụ 1
Innums = [2,2,3,2]
Out3
Ví dụ 2
Innums = [0,1,0,1,0,1,99]
Out99
Ý tưởng
Đếm từng bit mod 3 trên 32 bit; hoặc state machine ones/twos: ones giữ bit xuất hiện lần 1 mod 3, twos lần 2 mod 3.
Mã nguồn
solution.c C99 
/* State machine ones/twos — xem C++ */
class Solution:
    def singleNumber(self, nums: List[int]) -> int:
        ones = twos = 0
        for x in nums:
            ones = (ones ^ x) & ~twos
            twos = (twos ^ x) & ~ones
        return ones
class Solution {
public:
    int singleNumber(vector<int>& nums) {
        int ones = 0, twos = 0;
        for (int x : nums) {
            ones = (ones ^ x) & ~twos;
            twos = (twos ^ x) & ~ones;
        }
        return ones;
    }
};
