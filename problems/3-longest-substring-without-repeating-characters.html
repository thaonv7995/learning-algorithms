Longest Substring Without Repeating Characters

O(n) O(1) 39.1% AC
Mô tả
Cho chuỗi s. Tìm độ dài chuỗi con liên tiếp dài nhất không chứa ký tự lặp lại.
Ví dụ
Ví dụ 1
Ins = "abcabcbb"
Out3
Ví dụ 2
Ins = "bbbbb"
Out1
Ý tưởng
Sliding window [left, right] + map vị trí ký tự gần nhất. Khi s[right] trùng trong window → left = last[s[right]] + 1. Cập nhật maxLen.
Mã nguồn
solution.c C99 
/* Sliding window + charIndex[128] — xem C++ */
class Solution:
    def lengthOfLongestSubstring(self, s: str) -> int:
        last, left, best = {}, 0, 0
        for right, c in enumerate(s):
            if c in last and last[c] >= left:
                left = last[c] + 1
            last[c] = right
            best = max(best, right - left + 1)
        return best
class Solution {
public:
    int lengthOfLongestSubstring(string s) {
        vector<int> last(128, -1);
        int left = 0, best = 0;
        for (int r = 0; r < (int)s.size(); ++r) {
            if (last[s[r]] >= left) left = last[s[r]] + 1;
            last[s[r]] = r;
            best = max(best, r - left + 1);
        }
        return best;
    }
};
