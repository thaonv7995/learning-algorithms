Valid Palindrome

O(N) O(1)
Mô tả
Kiểm tra xem một chuỗi có phải là chuỗi đối xứng hay không, sau khi đã chuyển tất cả ký tự viết hoa thành viết thường và loại bỏ toàn bộ ký tự không phải chữ và số.
Ví dụ
Ví dụ 1
Ins = "A man, a plan, a canal: Panama"
Outtrue
Ý tưởng
Sử dụng hai con trỏ trái và phải (`left`, `right`) xuất phát từ hai biên chuỗi. Ta liên tục di chuyển hai con trỏ hướng vào tâm, bỏ qua tất cả ký tự đặc biệt không phải chữ và số thông qua hàm `isalnum()`. So sánh hai ký tự tại hai đầu sau khi chuẩn hóa về chữ thường `tolower()`. Nếu lệch nhau, trả về false.
Mã nguồn
solution.c C99 
// Solution in C (Valid Palindrome)
bool isPalindrome(char* s) {
    int left = 0, right = strlen(s) - 1;
    while (left < right) {
        while (left < right && !isalnum((unsigned char)s[left])) left++;
        while (left < right && !isalnum((unsigned char)s[right])) right--;
        if (tolower((unsigned char)s[left]) != tolower((unsigned char)s[right])) return false;
        left++; right--;
    }
    return true;
}
# Solution in Python (Valid Palindrome)
class Solution:
    def isPalindrome(self, s: str) -> bool:
        left, right = 0, len(s) - 1
        while left < right:
            while left < right and not s[left].isalnum():
                left += 1
            while left < right and not s[right].isalnum():
                right -= 1
            if s[left].lower() != s[right].lower():
                return False
            left += 1
            right -= 1
        return True
