Insertion Sort List

O(n²) O(1) 59.2% AC
Mô tả
Sắp xếp linked list bằng insertion sort: duyệt từng node, chèn vào vị trí đúng trong phần đã sắp (bên trái).
Ví dụ
Ví dụ 1
Inhead = [4,2,1,3]
Out[1,2,3,4]
Ví dụ 2
Inhead = [-1,5,3,4,0]
Out[-1,0,3,4,5]
Ý tưởng
Dummy head cho list đã sắp. Với mỗi node cur từ head gốc: tách cur, tìm prev sao cho prev.next.val <= cur.val, chèn cur sau prev.
Mã nguồn
solution.c C99 
/* Dummy head, chèn từng node — xem C++ */
class Solution:
    def insertionSortList(self, head: Optional[ListNode]) -> Optional[ListNode]:
        dummy = ListNode(0)
        cur = head
        while cur:
            nxt = cur.next
            prev = dummy
            while prev.next and prev.next.val < cur.val:
                prev = prev.next
            cur.next = prev.next
            prev.next = cur
            cur = nxt
        return dummy.next
class Solution {
public:
    ListNode* insertionSortList(ListNode* head) {
        ListNode dummy(0);
        ListNode* cur = head;
        while (cur) {
            ListNode* nxt = cur->next;
            ListNode* prev = &dummy;
            while (prev->next && prev->next->val < cur->val)
                prev = prev->next;
            cur->next = prev->next;
            prev->next = cur;
            cur = nxt;
        }
        return dummy.next;
    }
};
