Gray Code

O(2^n) O(2^n) 64.9% AC
Mô tả
Cho n ≥ 0. Trả về dãy Gray code gồm 2^n số nguyên từ 0..2^n-1 sao cho hai số liên tiếp chỉ khác đúng 1 bit (dãy có thể bắt đầu từ 0).
Ví dụ
Ví dụ 1
Inn = 2
Out[0,1,3,2]
Ví dụ 2
Inn = 1
Out[0,1]
Ý tưởng
Công thức: G(i) = i ^ (i >> 1). Hoặc đệ quy: l(list n-1) rồi reverse(list)+ (1<<(n-1)).
Mã nguồn
solution.c C99 
/* grayCode — i ^ (i>>1); xem C++ */
class Solution:
    def grayCode(self, n: int) -> List[int]:
        return [i ^ (i >> 1) for i in range(1 << n)]
class Solution {
public:
    vector<int> grayCode(int n) {
        int m=1<<n; vector<int> res(m);
        for (int i=0;i<m;i++) res[i]=i^(i>>1);
        return res;
    }
};
