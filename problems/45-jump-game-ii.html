Jump Game II

O(N) O(1) 42.9% AC
Mô tả
Từ index 0, nums[i] là bước nhảy tối đa tới i+j. Tìm số bước nhảy tối thiểu để tới index n−1 (luôn đến được).
Ví dụ
Ví dụ 1
Innums = [2,3,1,1,4]
Out2
Ví dụ 2
Innums = [2,3,0,1,4]
Out2
Ý tưởng
Greedy BFS: end = tầng hiện tại, far = xa nhất trong tầng. Duyệt i: cập nhật far = max(far, i+nums[i]); khi i==end thì steps++, end=far.
Mã nguồn
solution.c C99 
int jump(int* nums, int n) {
    int end = 0, far = 0, steps = 0;
    for (int i = 0; i < n - 1; i++) {
        if (i + nums[i] > far) far = i + nums[i];
        if (i == end) { steps++; end = far; }
    }
    return steps;
}
class Solution:
    def jump(self, nums: List[int]) -> int:
        end, far, steps = 0, 0, 0
        for i in range(len(nums)-1):
            far = max(far, i + nums[i])
            if i == end:
                steps += 1
                end = far
        return steps
class Solution {
public:
    int jump(vector<int>& nums) {
        int end=0, far=0, steps=0;
        for(int i=0;i<(int)nums.size()-1;i++){
            far=max(far,i+nums[i]);
            if(i==end){ steps++; end=far; }
        }
        return steps;
    }
};

  2. 2Duyệt chính — Duyệt i: cập nhật far = max; khi i==end thì steps++,…

Mỗi lần steps++ là một 'tầng' BFS — số tầng tối thiểu tới cuối.
