Nth Highest Salary

O(n log n) O(n) 39.2% AC
Mô tả
Viết function getNthHighestSalary(N INT) trả mức lương cao thứ N của bảng Employee; không có thì NULL.
Ví dụ
Ví dụ 1
InN=2, salaries [100,200,300]
Out200
Ví dụ 2
InN=1, salaries [100]
Out100
Ý tưởng
Function trả scalar subquery: SELECT DISTINCT salary ORDER BY salary DESC LIMIT 1 OFFSET N-1.
Mã nguồn
solution.c C99 
/* SQL */
CREATE FUNCTION getNthHighestSalary(N INT) RETURNS INT
BEGIN
  SET N = N - 1;
  RETURN (
    SELECT DISTINCT salary FROM Employee
    ORDER BY salary DESC LIMIT 1 OFFSET N
  );
END;
