Fibonacci Number

O(n) O(1) 74.1% AC
Mô tả
Tính số Fibonacci thứ n (F0=0, F1=1, Fn=F(n-1)+F(n-2)). 0 ≤ n ≤ 30.
Ví dụ

Ý tưởng
Lặp từ 2 đến n: cập nhật (a,b) = (b, a+b) — O(1) space.
Mã nguồn
solution.c C99 
/* Rolling fib — xem C++ */
class Solution:
    def fib(self, n: int) -> int:
        if n <= 1:
            return n
        a, b = 0, 1
        for _ in range(2, n + 1):
            a, b = b, a + b
        return b
class Solution {
public:
    int fib(int n) {
        if (n <= 1) return n;
        int a = 0, b = 1;
        for (int i = 2; i <= n; i++) {
            int c = a + b;
            a = b;
            b = c;
        }
        return b;
    }
};
