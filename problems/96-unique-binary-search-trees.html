Unique Binary Search Trees

O(n²) O(n) 63.7% AC
Mô tả
Cho số nguyên n ≥ 1. Trả về số cây BST khác nhau có thể tạo từ n nút có giá trị phân biệt 1..n. Hai cây khác nhau nếu cấu trúc liên kết khác (không chỉ khác giá trị nhãn).
Ví dụ
Ví dụ 1
Inn = 3
Out5
Ví dụ 2
Inn = 1
Out1
Ý tưởng
Số Catalan G(n): chọn root i (1..n), cây con trái có i−1 nút, phải có n−i nút → G(n) = Σ G(i−1)·G(n−i), G(0)=1. DP bottom-up hoặc công thức trực tiếp.
Mã nguồn
solution.c C99 
/* DP Catalan G(n) — xem C++ */
class Solution:
    def numTrees(self, n: int) -> int:
        dp = [0] * (n + 1)
        dp[0] = 1
        for nodes in range(1, n + 1):
            for root in range(1, nodes + 1):
                dp[nodes] += dp[root - 1] * dp[nodes - root]
        return dp[n]
class Solution {
public:
    int numTrees(int n) {
        vector<long long> dp(n + 1, 0);
        dp[0] = 1;
        for (int nodes = 1; nodes <= n; ++nodes) {
            for (int root = 1; root <= nodes; ++root) {
                dp[nodes] += dp[root - 1] * dp[nodes - root];
            }
        }
        return (int)dp[n];
    }
};
