Balanced Binary Tree

O(n) O(h) 58.4% AC
Mô tả
Cho root cây nhị phân. Kiểm tra cây có cân bằng không: |chiều cao cây trái − chiều cao cây phải| ≤ 1 tại mọi nút.
Ví dụ
Ví dụ 1
Inroot = [3,9,20,null,null,15,7]
Outtrue
Ví dụ 2
Inroot = [1,2,2,3,3,null,null,4,4]
Outfalse
Ví dụ 3
Inroot = []
Outtrue
Ý tưởng
DFS bottom-up: hàm depth trả chiều cao, hoặc -1 nếu mất cân bằng. |L−R|>1 → propagate -1 lên.
Mã nguồn
solution.c C99 
/* DFS trả height hoặc -1 — xem C++ */
class Solution:
    def isBalanced(self, root: Optional[TreeNode]) -> bool:
        def depth(node):
            if not node: return 0
            L, R = depth(node.left), depth(node.right)
            if L < 0 or R < 0 or abs(L - R) > 1: return -1
            return 1 + max(L, R)
        return depth(root) >= 0
class Solution {
    int depth(TreeNode* nd) {
        if (!nd) return 0;
        int L = depth(nd->left), R = depth(nd->right);
        if (L < 0 || R < 0 || abs(L - R) > 1) return -1;
        return 1 + max(L, R);
    }
public:
    bool isBalanced(TreeNode* root) {
        return depth(root) >= 0;
    }
};
