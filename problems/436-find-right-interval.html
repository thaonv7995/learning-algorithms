Find Right Interval

O(n log n) O(n) 55.9% AC
Mô tả
Cho intervals [start,end]. Với mỗi i, tìm j nhỏ nhất sao start_j >= end_i (right interval); không có thì -1.
Ví dụ
Ví dụ 1
Inintervals = [[3,4],[2,3],[1,2]]
Out[-1,0,1]
Ví dụ 2
Inintervals = [[1,4],[2,3],[3,4]]
Out[-1,2,-1]
Ý tưởng
Sort (start, original index); với mỗi end_i binary search lower_bound start >= end_i.
Mã nguồn

int* findRightInterval(int** iv, int n, int* cs, int* rs) { *rs=n; return NULL; }
class Solution:
    def findRightInterval(self, intervals: List[List[int]]) -> List[int]:
        starts = sorted((s, i) for i, (s, _) in enumerate(intervals))
        ans = []
        for _, e in intervals:
            lo, hi = 0, len(starts)
            while lo < hi:
                mid = (lo + hi) // 2
                if starts[mid][0] >= e: hi = mid
                else: lo = mid + 1
            ans.append(starts[lo][1] if lo < len(starts) else -1)
        return ans
class Solution {
public:
    vector<int> findRightInterval(vector<vector<int>>& v) {
        vector<pair<int,int>> st;
        for(int i=0;i<(int)v.size();i++) st.push_back({v[i][0],i});
        sort(st.begin(), st.end());
        vector<int> ans;
        for(auto& p:v){
            int e=p[1];
            auto it=lower_bound(st.begin(), st.end(), make_pair(e,0));
            ans.push_back(it==st.end()?-1:it->second);
        }
        return ans;
    }
};
