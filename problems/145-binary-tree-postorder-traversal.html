Binary Tree Postorder Traversal

O(n) O(h) 78.2% AC
Mô tả
Cho root cây nhị phân. Trả thứ tự duyệt postorder (trái → phải → node).
Ví dụ
Ví dụ 1
Inroot = [1,null,2,3]
Out[3,2,1]
Ví dụ 2
Inroot = [1,2,3,4,5,null,8,null,null,6,7,9]
Out[4,6,7,5,2,9,8,3,1]
Ví dụ 3
Inroot = []
Out[]
Ý tưởng
Đệ quy: postorder trái, phải, rồi thêm root.val. Iterative: stack + visited flag, hoặc reverse preorder (root-right-left) rồi đảo kết quả.
Mã nguồn
solution.c C99 
/* Stack root-left-right rồi reverse — xem C++ */
class Solution:
    def postorderTraversal(self, root: Optional[TreeNode]) -> List[int]:
        res, st = [], []
        if root:
            st.append(root)
        while st:
            node = st.pop()
            res.append(node.val)
            if node.left:
                st.append(node.left)
            if node.right:
                st.append(node.right)
        return res[::-1]
class Solution {
public:
    vector<int> postorderTraversal(TreeNode* root) {
        vector<int> res;
        if (!root) return res;
        stack<TreeNode*> st;
        st.push(root);
        while (!st.empty()) {
            TreeNode* node = st.top(); st.pop();
            res.push_back(node->val);
            if (node->left) st.push(node->left);
            if (node->right) st.push(node->right);
        }
        reverse(res.begin(), res.end());
        return res;
    }
};
