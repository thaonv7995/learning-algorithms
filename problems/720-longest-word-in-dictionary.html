Longest Word in Dictionary

O(n * L^2) O(n * L) 54.8% AC
Mô tả
Cho mảng chuỗi words. Trả chuỗi dài nhất trong từ điển mà mọi tiền tố (prefix) của nó cũng nằm trong words. Nếu nhiều đáp án, trả chuỗi nhỏ nhất theo thứ tự từ điển.
Ví dụ
Ví dụ 1
Inwords = ["w","wo","wor","worl","world"]
Out"world"
Ví dụ 2
Inwords = ["a","banana","app","appl","ap","apply","apple"]
Out"apple"
Ví dụ 3
Inwords = ["m","mo","moc","moch","mocha","l","la","lat"]
Out"mocha"
Ý tưởng
Sort words. Duyệt từng word: kiểm tra mọi prefix có trong set words; nếu đủ và dài hơn (hoặc bằng nhưng lex nhỏ hơn) thì cập nhật ans.

}
class Solution:
    def longestWord(self, words: List[str]) -> str:
        words.sort()
        st = set(words)
        ans = ""
        for w in words:
            ok = True
            for i in range(1, len(w)):
                if w[:i] not in st:
                    ok = False; break
            if ok and (len(w) > len(ans) or (len(w) == len(ans) and w < ans)):
                ans = w
        return ans
class Solution {
public:
    string longestWord(vector<string>& words) {
        sort(words.begin(), words.end());
        unordered_set<string> st(words.begin(), words.end());
        string ans;
        for (auto& w : words) {
            bool ok = true;
            for (int i = 1; i < (int)w.size(); i++)
                if (!st.count(w.substr(0, i))) { ok = false; break; }
            if (ok && (w.size() > ans.size() || (w.size() == ans.size() && w < ans))) ans = w;
        }
        return ans;
    }
};
