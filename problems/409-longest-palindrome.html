Longest Palindrome

O(n) O(1) 56% AC
Mô tả
Cho chuỗi s gồm chữ thường. Dùng các ký tự (hoán vị) để ghép palindrome dài nhất. Trả độ dài palindrome tối đa.
Ví dụ
Ví dụ 1
Ins = "abccccdd"
Out7
Ví dụ 2
Ins = "a"
Out1
Ví dụ 3
Ins = "bb"
Out2
Ý tưởng
Đếm tần suất 26 chữ; mỗi chữ c chẵn count cộng hết, lẻ cộng count-1; nếu có bất kỳ count lẻ thì +1 cho ký tự giữa.
Mã nguồn
solution.c C99 
int longestPalindrome(char* s) {
    int cnt[26]={0};
    for(;*s;s++) cnt[*s-'a']++;
    int ans=0, odd=0;
    for(int i=0;i<26;i++){ ans+=cnt[i]/2*2; if(cnt[i]&1) odd=1; }
    return ans+odd;
}
class Solution:
    def longestPalindrome(self, s: str) -> int:
        cnt = Counter(s)
        ans = odd = 0
        for c in cnt.values():
            ans += c // 2 * 2
            odd |= c & 1
        return ans + odd
