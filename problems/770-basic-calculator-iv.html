Basic Calculator IV

O(n) O(n) 49.8% AC
Mô tả
Parse biểu thức: số, biến, (expr), toán tử +-* và ngoặc. Trả vector string dạng ["2*a","3"] sau khi gom hạng tương tự (Basic Calculator IV).
Ví dụ
Ví dụ 1
Inexpression = "e + 8 - a + 5", evalvars = ["e"], evalints = [1]
Out["-1*a","14"]
Ví dụ 2
Inexpression = "e - 8 + temperature - pressure", evalvars = ["e","temperature"], evalints = [1,12]
Out["-1*pressure","5"]
Ý tưởng
Parse thành map var→coeff và constant; thay evalvars; format mọi term sort lex.
