Binary Search Tree Iterator

O(1) amortized next O(h) 76.5% AC
Mô tả
Thiết kế iterator inorder cho BST: next() trả phần tử kế tiếp theo thứ tự tăng dần, hasNext() kiểm tra còn phần tử. average O(1) cho next/hasNext.
Ví dụ
Ví dụ 1
InBST [7,3,15,null,null,9,20], next()×3
Out[3,7,9]
Ví dụ 2
Innext() tiếp, hasNext()
Out[15,true]
Ý tưởng
Stack: pushLeft(root) đẩy cả chuỗi trái. next() pop, nếu node.right thì pushLeft(right). hasNext() = stack không rỗng.
Mã nguồn
solution.c C99 
typedef struct { struct TreeNode** st; int top, cap; } BSTIter;
static void pushLeft(BSTIter* it, struct TreeNode* n) {
    while (n) {
        if (++it->top >= it->cap) { it->cap*=2; it->st=realloc(it->st,it->cap*sizeof(*it->st)); }
        it->st[it->top]=n; n=n->left;
    }
}
BSTIter* bSTIteratorCreate(struct TreeNode* root) {
    BSTIter* it=malloc(sizeof(BSTIter)); it->cap=16; it->top=-1;
    it->st=malloc(it->cap*sizeof(*it->st)); pushLeft(it,root); return it;
}
int bSTIteratorNext(BSTIter* it) {
    struct TreeNode* n=it->st[it->top--]; pushLeft(it,n->right); return n->val;
}
bool bSTIteratorHasNext(BSTIter* it) { return it->top>=0; }
class BSTIterator:
    def __init__(self, root: Optional[TreeNode]):
        self.st = []
        self._left(root)
    def _left(self, node):
        while node:
            self.st.append(node)
            node = node.left
    def next(self) -> int:
        node = self.st.pop()
        self._left(node.right)
        return node.val
    def hasNext(self) -> bool:
        return bool(self.st)
class BSTIterator {
    stack<TreeNode*> st;
    void pushLeft(TreeNode* n) {
        while (n) { st.push(n); n = n->left; }
    }
public:
    BSTIterator(TreeNode* root) { pushLeft(root); }
    int next() {
        TreeNode* n = st.top(); st.pop();
        pushLeft(n->right);
        return n->val;
    }
    bool hasNext() { return !st.empty(); }
};
Stack O(1) O(h)
  1. 1Khởi tạo — Stack: pushLeft đẩy cả chuỗi trái.
  2. 2Duyệt chính — next pop, nếu node.right thì pushLeft.
  3. 3Cập nhật — hasNext = stack không rỗng.

Stack giữ đúng tiền tố inorder chưa xuất; mỗi node push/pop đúng một lần.
