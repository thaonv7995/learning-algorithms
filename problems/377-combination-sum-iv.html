Combination Sum IV

O(target×n) O(target) 55.1% AC
Mô tả
Cho mảng nums số nguyên dương distinct và target. Đếm số tổ hợp có thứ tự (permutation) các số trong nums cộng bằng target.
Ví dụ
Ví dụ 1
Innums = [1,2,3], target = 4
Out7
Ví dụ 2
Innums = [9], target = 3
Out0
Ý tưởng
DP: dp[0]=1; với t từ 1..target, dp[t]+=dp[t-num] với mọi num≤t. Thứ tự outer t quan trọng (permutation).
Mã nguồn
solution.c C99 
int combinationSum4(int* nums, int n, int target) {
    unsigned* dp = calloc(target + 1, sizeof(unsigned)); dp[0] = 1;
    for (int t = 1; t <= target; t++)
        for (int i = 0; i < n; i++)
            if (nums[i] <= t) dp[t] += dp[t - nums[i]];
    int ans = (int)dp[target]; free(dp); return ans;
}
class Solution:
    def combinationSum4(self, nums: List[int], target: int) -> int:
        dp = [0] * (target + 1)
        dp[0] = 1
        for t in range(1, target + 1):
            for x in nums:
                if x <= t:
                    dp[t] += dp[t - x]
        return dp[target]
class Solution {
public:

    }
};
