Reverse Integer

O(log N) O(1) 31.9% AC
Mô tả
Đảo ngược các chữ số của số nguyên 32-bit signed x. Trả 0 nếu reverse overflow.
Ví dụ
Ví dụ 1
Inx = 123
Out321
Ví dụ 2
Inx = -123
Out-321
Ví dụ 3
Inx = 120
Out21
Ý tưởng
Lấy digit cuối bằng x % 10, x = x / 10, result = result * 10 + digit. Kiểm tra overflow trước khi nhân: nếu result > INT_MAX/10 hoặc bằng và digit > 7 thì return 0.
Mã nguồn
solution.c C99 
int reverse(int x) {
    long res = 0;
    while (x) {
        res = res * 10 + x % 10;
        x /= 10;
    }
    return (res < -2147483648L || res > 2147483647L) ? 0 : (int)res;
}
class Solution:
    def reverse(self, x: int) -> int:
        sign = -1 if x < 0 else 1
        r = int(str(abs(x))[::-1])
        r *= sign
        return r if -2**31 <= r <= 2**31 - 1 else 0
