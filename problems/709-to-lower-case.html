To Lower Case

O(n) O(n) 84.9% AC
Mô tả
Cho chuỗi s. Trả bản sao s với mọi chữ hoa chuyển thành chữ thường (ASCII).
Ví dụ
Ví dụ 1
Ins = "Hello"
Out"hello"
Ví dụ 2
Ins = "here"
Out"here"
Ví dụ 3
Ins = "LOVELY"
Out"lovely"
Ý tưởng
Duyệt từng ký tự: nếu A-Z thì cộng 32 (hoặc |32) để thành a-z.
Mã nguồn
solution.c C99 
char* toLowerCase(char* s) {
    for (int i = 0; s[i]; i++)
        if (s[i] >= 'A' && s[i] <= 'Z') s[i] += 32;
    return s;
}
class Solution:
    def toLowerCase(self, s: str) -> str:
        return "".join(ch.lower() for ch in s)
class Solution {
public:
    string toLowerCase(string s) {
        for (char& c : s)
            if (c >= 'A' && c <= 'Z') c += 32;
        return s;
    }
};
