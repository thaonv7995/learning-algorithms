Reverse String II

O(n) O(1) 53.8% AC
Mô tả
Cho s và k. Đảo chuỗi theo khối 2k: mỗi 2k ký tự đảo k ký tự đầu; đoạn cuối <2k thì đảo k đầu (hoặc hết nếu ngắn hơn).
Ví dụ
Ví dụ 1
Ins = "abcdefg", k = 2
Out"bacdfeg"
Ví dụ 2
Ins = "abcd", k = 2
Out"bacd"
Ý tưởng
Duyệt i += 2k; reverse s[i..i+k-1] bằng two-pointer.
Mã nguồn
solution.c C99 
/* Reverse every 2k block — xem C++ */
class Solution:
    def reverseStr(self, s: str, k: int) -> str:
        a = list(s)
        for i in range(0, len(a), 2 * k):
            a[i:i+k] = reversed(a[i:i+k])
        return "".join(a)
class Solution {
public:
    string reverseStr(string s, int k) {
        for (int i = 0; i < (int)s.size(); i += 2 * k) {
            reverse(s.begin() + i, s.begin() + min(i + k, (int)s.size()));
        }
        return s;
    }
};
