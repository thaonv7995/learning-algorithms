Minimum Number of Arrows to Burst Balloons

O(n log n) O(1) 61.5% AC
Mô tả
Mũi tên bắn balloon intervals [start,end]. Một mũi tên bay thẳng x bắn mọi balloon có start≤x≤end. Min số mũi tên.
Ví dụ
Ví dụ 1
Inpoints = [[10,16],[2,8],[1,6],[7,12]]
Out2
Ví dụ 2
Inpoints = [[1,2],[3,4],[5,6],[7,8]]
Out4
Ý tưởng
Sort theo end; greedy: bắn tại end balloon đầu chưa bị bắn, skip balloon overlap.

class Solution:
    def findMinArrowShots(self, points: List[List[int]]) -> int:
        points.sort(key=lambda x: x[1])
        ans, end = 0, float("-inf")
        for s, e in points:
            if s > end: ans += 1; end = e
        return ans
class Solution {
public:
