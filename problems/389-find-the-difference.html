Find the Difference

O(n) O(1) 60.3% AC
Mô tả
Chuỗi t gồm s cộng thêm đúng một ký tự ngẫu nhiên. Trả ký tự được thêm.
Ví dụ
Ví dụ 1
Ins = "abcd", t = "abcde"
Out"e"
Ví dụ 2
Ins = "", t = "y"
Out"y"
Ý tưởng
XOR tất cả ký tự s và t — ký tự lẻ còn lại là đáp án. Hoặc count frequency.
Mã nguồn
solution.c C99 
char findTheDifference(char* s, char* t) {
    char x = 0;
    for (; *s; s++) x ^= *s;
    for (; *t; t++) x ^= *t;
    return x;
}
class Solution:
    def findTheDifference(self, s: str, t: str) -> str:
        x = 0
        for c in s + t:
            x ^= ord(c)
        return chr(x)
class Solution {
public:
    char findTheDifference(string s, string t) {
        char x = 0;
        for (char c : s) x ^= c;
        for (char c : t) x ^= c;
        return x;
    }
};
