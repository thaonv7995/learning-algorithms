Diagonal Traverse

O(mn) O(1) 67.2% AC
Mô tả
Trả các phần tử matrix theo thứ tự diagonal traverse (↗ rồi ↙ xen kẽ).
Ví dụ
Ví dụ 1
Inmat = [[1,2,3],[4,5,6],[7,8,9]]
Out[1,2,4,7,5,3,6,8,9]
Ví dụ 2
Inmat = [[1,2],[3,4]]
Out[1,2,3,4]
Ý tưởng
Simulate direction: start (0,0); if top row or even diagonal go down-left else up-right.

class Solution:
    def findDiagonalOrder(self, mat: List[List[int]]) -> List[int]:
        if not mat: return []
        m, n = len(mat), len(mat[0])
        ans = []
        r = c = 0
        for _ in range(m*n):
            ans.append(mat[r][c])
            if (r+c) % 2 == 0:
                if c == n-1: r += 1
                elif r == 0: c += 1
                else: r -= 1; c += 1
            else:
                if r == m-1: c += 1
                elif c == 0: r += 1
                else: r += 1; c -= 1
        return ans
