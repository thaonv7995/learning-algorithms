Climbing Stairs

O(N) O(1) 54.1% AC
Mô tả
Leo n bậc thang, mỗi bước 1 hoặc 2 bậc. Đếm số cách khác nhau lên đỉnh.
Ví dụ
Ví dụ 1
Inn = 2
Out2
Ví dụ 2
Inn = 3
Out3
Ý tưởng
dp[i] = dp[i-1] + dp[i-2] (Fibonacci). Tối ưu 2 biến a,b cuộn.
Mã nguồn
solution.c C99 
int climbStairs(int n) {
    if (n <= 2) return n;
    int a = 1, b = 2;
    for (int i = 3; i <= n; i++) { int t = a + b; a = b; b = t; }
    return b;
}
class Solution:
    def climbStairs(self, n: int) -> int:
        if n <= 2: return n
        a, b = 1, 2
        for _ in range(3, n + 1):
            a, b = b, a + b
        return b
