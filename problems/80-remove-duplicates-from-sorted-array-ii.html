Remove Duplicates from Sorted Array II

O(n) O(1) 64.7% AC
Mô tả
Mảng nums đã sort không giảm. Sửa in-place sao cho mỗi phần tử xuất hiện tối đa 2 lần. Trả về k = số phần tử hợp lệ; phần tử nums[0..k-1] là kết quả.
Ví dụ
Ví dụ 1
Innums = [1,1,1,2,2,3]
Out5, nums = [1,1,2,2,3]
Ví dụ 2
Innums = [0,0,1,1,1,1,2,3,3]
Out7, nums = [0,0,1,1,2,3,3]
Ý tưởng
Two pointers: k là vị trí ghi tiếp. Cho phép ghi nums[i] nếu k<2 hoặc nums[i] <= nums[k-2] (tức chưa quá 2 bản).
Mã nguồn
solution.c C99 
int removeDuplicates(int* nums, int n) {
    int k=0;
    for (int i=0;i<n;i++)
        if (k<2 || nums[i]<=nums[k-2]) nums[k++]=nums[i];
    return k;
}
class Solution:
    def removeDuplicates(self, nums: List[int]) -> int:
        k = 0
        for x in nums:
            if k < 2 or x <= nums[k - 2]:
                nums[k] = x; k += 1
        return k
class Solution {
public:
    int removeDuplicates(vector<int>& a) {
        int k=0;
        for (int x: a)
            if (k<2 || x<=a[k-2]) a[k++]=x;
        return k;
    }
};
