Longest Consecutive Sequence

O(n) O(n) 47.1% AC
Mô tả
Cho mảng nums không sắp xếp. Tìm độ dài dãy số nguyên liên tiếp dài nhất (không cần liên tiếp trong mảng).
Ví dụ
Ví dụ 1
Innums = [100,4,200,1,3,2]
Out4
Ví dụ 2
Innums = [0,3,7,2,5,8,4,6,0,1]
Out9
Ý tưởng
Đưa nums vào set. Chỉ bắt đầu từ số n nếu n-1 không có trong set (đầu dãy). Từ n đếm n, n+1, ... trong set.
Mã nguồn
solution.c C99 
/* Set + chỉ start đầu dãy — xem C++ */
class Solution:
    def longestConsecutive(self, nums: List[int]) -> int:
        s = set(nums)
        best = 0
        for n in s:
            if n - 1 not in s:
                cur, length = n, 0
                while cur in s:
                    length += 1
                    cur += 1
                best = max(best, length)
        return best
class Solution {
public:
    int longestConsecutive(vector<int>& nums) {
        unordered_set<int> s(nums.begin(), nums.end());
        int best = 0;
        for (int n : s) {
            if (s.count(n - 1)) continue;
            int cur = n, len = 0;
            while (s.count(cur)) { len++; cur++; }
            best = max(best, len);
        }
        return best;
    }
};
