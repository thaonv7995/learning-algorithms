1-bit and 2-bit Characters

O(n) O(1) 49.6% AC
Mô tả
Mảng bits gồm 0 và 1 mô tả mã hóa: 0 là ký tự 1-bit; 10 hoặc 11 là ký tự 2-bit. bit[0] luôn là 1-bit. Kiểm tra mảng có phải chuỗi mã hóa hợp lệ không (ăn hết mọi bit).
Ví dụ
Ví dụ 1
Inbits = [1,0,0]
Outtrue
Ví dụ 2
Inbits = [1,1,1,0]
Outfalse
Ví dụ 3
Inbits = [1,0,1]
Outtrue
Ý tưởng
Duyệt i=0: nếu bits[i]==0 nhảy 1; else nếu i+1 hợp lệ và bits[i+1] in {0,1} nhảy 2; else false. Cuối i==n → true.
Mã nguồn
solution.c C99 
bool isOneBitCharacter(int* bits, int n) {
    int i = 0;
    while (i < n) {
        if (bits[i] == 0) i++;
        else if (i + 1 < n) i += 2;
        else return false;
    }
    return true;
}
class Solution:
    def isOneBitCharacter(self, bits: List[int]) -> bool:
        i, n = 0, len(bits)
        while i < n:
            if bits[i] == 0:
                i += 1
            elif i + 1 < n:
                i += 2
            else:
                return False
        return True
