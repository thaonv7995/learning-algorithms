Verify Preorder Serialization of a Binary Tree

O(n) O(1) 47.4% AC
Mô tả
Chuỗi preorder serialization dùng số nút và '#' đánh dấu null. Kiểm tra chuỗi có mô tả cây nhị phân hợp lệ không.
Ví dụ
Ví dụ 1
Inpreorder = "9,3,4,#,#,1,#,#,6,#,#,#"
Outtrue
Ví dụ 2
Inpreorder = "1,#"
Outfalse
Ví dụ 3
Inpreorder = "9,#,#,1"
Outfalse
Ý tưởng
Slot counting: bắt đầu slots=1; mỗi token tốn 1 slot; '#' thêm 1 slot, số thêm 2 slot; hết chuỗi slots==0.
Mã nguồn
solution.c C99 
bool isValidSerialization(char* preorder) {
    int slots = 1;
    char* tok = strtok(preorder, ",");
    while (tok) {
        slots--;
        if (slots < 0) return false;
        if (tok[0] != '#' || tok[1]) slots += 2;
        tok = strtok(NULL, ",");
    }
    return slots == 0;
}
class Solution:
    def isValidSerialization(self, preorder: str) -> bool:
        slots = 1
        for node in preorder.split(','):
            slots -= 1
            if slots < 0:
                return False
            if node != '#':
                slots += 2
        return slots == 0
class Solution {
public:
    bool isValidSerialization(string preorder) {
        int slots = 1;
        string cur;
        for (char c : preorder + ',') {
            if (c == ',') {
                slots--;
                if (slots < 0) return false;
                if (cur != "#") slots += 2;
                cur.clear();
            } else cur.push_back(c);
        }
        return slots == 0;
    }
};
