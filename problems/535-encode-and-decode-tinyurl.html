Encode and Decode TinyURL

O(1) encode/decode O(n) 86.6% AC
Mô tả
Thiết kế hệ thống rút gọn URL: encode(longUrl) trả shortUrl ngẫu nhiên; decode(shortUrl) trả longUrl gốc.
Ví dụ
Ví dụ 1
Inencode("https://example.com/long/path")
Outshort key vd. "http://tinyurl.com/0"
Ví dụ 2
Indecode short key
OutlongUrl ban đầu
Ý tưởng
Map bidirectional: long→short và short→long. Sinh key ngẫu nhiên hoặc tăng dần counter cho path.
Mã nguồn
solution.c C99 
/* Bidirectional hash map — xem C++ */
class Codec:
    def __init__(self):
        self.l2s = {}
        self.s2l = {}
        self.base = "http://tinyurl.com/"
    def encode(self, longUrl: str) -> str:
        if longUrl in self.l2s:
            return self.l2s[longUrl]
        key = str(len(self.l2s))
        short = self.base + key
        self.l2s[longUrl] = short
        self.s2l[short] = longUrl
        return short
    def decode(self, shortUrl: str) -> str:
        return self.s2l[shortUrl]
class Codec {
    unordered_map<string,string> l2s, s2l;
    string base = "http://tinyurl.com/";
public:
    string encode(string longUrl) {
        if (l2s.count(longUrl)) return l2s[longUrl];
        string key = to_string(l2s.size());
        string shortUrl = base + key;
        l2s[longUrl] = shortUrl;
        s2l[shortUrl] = longUrl;
        return shortUrl;
    }
    string decode(string shortUrl) { return s2l[shortUrl]; }
};
