All Paths From Source to Target

O(2^n) O(n) 83.6% AC
Mô tả
DAG với n node 0..n-1, graph[u] = neighbors. Liệt kê mọi path từ 0 tới n-1 (sorted lex).
Ví dụ
Ví dụ 1
Ingraph = [[1,2],[3],[3],[]]
Out[[0,1,3],[0,2,3]]
Ví dụ 2
Ingraph = [[4,3,1],[3,2,4],[3],[4],[]]
Out[[0,4],[0,3,4],[0,1,3,4],[0,1,2,3,4],[0,1,4]]
Ý tưởng
DFS backtrack từ 0, push path khi tới n-1.

class Solution:
    def allPathsSourceTarget(self, graph: List[List[int]]) -> List[List[int]]:
        n = len(graph)
        ans = []
        path = [0]
        def dfs(u):
            if u == n - 1:
                ans.append(path[:]); return
            for v in graph[u]:
                path.append(v); dfs(v); path.pop()
        dfs(0)
        return ans
class Solution {
public:

        int n=g.size(); vector<vector<int>> ans; vector<int> path={0};
        function<void(int)> dfs=[&](int u){
            if(u==n-1){ ans.push_back(path); return; }
            for(int v:g[u]){ path.push_back(v); dfs(v); path.pop_back(); }
        };
        dfs(0); return ans;
    }
