Permutation Sequence

O(N²) O(N) 53.1% AC
Mô tả
Trả hoán vị thứ k (1-indexed) của tập {1..n} theo thứ tự từ điển.
Ví dụ
Ví dụ 1
Inn = 3, k = 3
Out"213"
Ví dụ 2
Inn = 4, k = 9
Out"2314"
Ý tưởng
Tính factorial. Mỗi vị trí: idx = (k-1)/fact[n-1-i], chọn số thứ idx từ list còn lại, k -= idx*fact[...].
Mã nguồn
solution.c C99 
/* getPermutation — factorial expansion; xem C++ */
class Solution:
    def getPermutation(self, n: int, k: int) -> str:
        fact=[1]*n
        for i in range(1,n): fact[i]=fact[i-1]*i
        nums=list(range(1,n+1)); k-=1; res=[]
        for i in range(n,0,-1):
            idx=k//fact[i-1]; res.append(str(nums.pop(idx))); k-=idx*fact[i-1]
        return ''.join(res)
class Solution {
public:
    string getPermutation(int n, int k) {
        vector<int> fact(n,1), nums;
        for(int i=1;i<n;i++) fact[i]=fact[i-1]*i;
        for(int i=1;i<=n;i++) nums.push_back(i);
        k--;
        string res;
        for(int i=n;i>=1;i--){
            int idx=k/fact[i-1];
            res+=char('0'+nums[idx]); nums.erase(nums.begin()+idx);
            k-=idx*fact[i-1];
        }
        return res;
    }
};
