Maximum Length of Pair Chain

O(n log n) O(1) 61.8% AC
Mô tả
Cho pairs [a,b] với a<b. Chuỗi pair chain nếu b_prev < a_next. Tìm độ dài chain dài nhất.
Ví dụ
Ví dụ 1
Inpairs = [[1,2],[2,3],[3,4]]
Out2
Ví dụ 2
Inpairs = [[1,2],[7,8],[4,5]]
Out2
Ý tưởng
Sort theo end b; greedy chọn nếu a>prev_end.

class Solution:
    def findLongestChain(self, pairs: List[List[int]]) -> int:
        pairs.sort(key=lambda x:x[1]); end=-10**9; ans=0
        for a,b in pairs:
            if a>end: ans+=1; end=b
        return ans
class Solution {
public:
    int findLongestChain(vector<vector<int>>& p) {
        sort(p.begin(),p.end(),[](auto&a,auto&b){return a[1]<b[1];});
        int end=INT_MIN, ans=0;
        for(auto& v:p) if(v[0]>end){ ans++; end=v[1]; }
        return ans;
    }
};
