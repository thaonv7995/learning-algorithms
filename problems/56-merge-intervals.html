Merge Intervals

O(N log N) O(N) 51.8% AC
Mô tả
Gộp các khoảng [start,end] chồng lấn thành khoảng không overlap.
Ví dụ
Ví dụ 1
Inintervals = [[1,3],[2,6],[8,10],[15,18]]
Out[[1,6],[8,10],[15,18]]
Ví dụ 2
Inintervals = [[1,4],[4,5]]
Out[[1,5]]
Ý tưởng
Sort theo start. Duyệt: nếu start ≤ end của khoảng cuối → merge max end; else push khoảng mới.
Mã nguồn
solution.c C99 
/* merge intervals — sort + linear merge; xem C++ */
class Solution:
    def merge(self, intervals: List[List[int]]) -> List[List[int]]:
        intervals.sort(key=lambda x: x[0])
        res = [intervals[0]]
        for s, e in intervals[1:]:
            if s <= res[-1][1]: res[-1][1] = max(res[-1][1], e)
            else: res.append([s, e])
        return res
class Solution {
public:
    vector<vector<int>> merge(vector<vector<int>>& in) {
        sort(in.begin(), in.end());
        vector<vector<int>> res={in[0]};
        for(int i=1;i<(int)in.size();i++){
            if(in[i][0]<=res.back()[1]) res.back()[1]=max(res.back()[1],in[i][1]);
            else res.push_back(in[i]);
        }
        return res;
    }
};
